## Βήμα 3: Ορίστε τις ανοχές χρησιμοποιώντας το FileGdbOptions
 Καθορίστε τις ανοχές για το επίπεδο Αρχείο GDB χρησιμοποιώντας το`FileGdbOptions` τάξη:
```csharp
var options = new FileGdbOptions
{
    XYTolerance = 0.001,
    ZTolerance = 0.1,
    MTolerance = 0.1,
};
```
