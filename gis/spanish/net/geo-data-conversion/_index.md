## Preguntas frecuentes

**P: ¿Puedo usar estas conversiones en un entorno de producción?**  
R: Sí. Una licencia comercial de Aspose.GIS elimina todas las limitaciones de prueba y brinda soporte técnico completo.

**P: ¿Qué runtimes de .NET son compatibles?**  
R: La biblioteca funciona con .NET Framework 4.6+, .NET Core 3.1+, .NET 5 y .NET 6.

**P: ¿Necesito instalar algún software GIS nativo?**  
R: No. Aspose.GIS es una biblioteca .NET puramente administrada; no se requieren dependencias externas.

**P: ¿Qué tan grande puede ser un archivo que puedo convertir?**  
R: Los archivos de hasta varios cientos de megabytes se manejan cómodamente; para conjuntos de datos muy grandes considere las APIs de streaming proporcionadas por Aspose.GIS.

**P: ¿Hay alguna forma de preservar los sistemas de referencia de coordenadas (CRS) durante la conversión?**  
R: Sí. La API retiene automáticamente la información del CRS a menos que reproyectes explícitamente los datos.
