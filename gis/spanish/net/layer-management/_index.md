## Preguntas frecuentes

**Q:** ¿Cómo puedo **crear nuevos shapefile** programáticamente?  
**A:** Usa la clase `Shapefile` en Aspose.GIS, especifica el tipo de geometría, agrega campos y luego escribe las entidades. El tutorial vinculado “Crear nuevo Shapefile” muestra el código exacto.

**Q:** ¿Cuál es la diferencia entre un File GDB y un shapefile?  
**A:** Un File Geodatabase (File GDB) almacena datos en una estructura basada en carpetas con soporte de esquema más rico, mientras que un shapefile es un formato simple y ampliamente compatible que consiste en varios archivos de longitud fija.

**Q:** ¿Cómo puedo **filtrar entidades por atributo** de manera eficiente?  
**A:** Construye un objeto `Query` con una cláusula WHERE (p.ej., `SELECT * FROM layer WHERE Population > 1000`). La guía “Filtrar entidades por atributo” te guía a través de la API.

**Q:** ¿Puedo convertir un shapefile a GeoJSON sin archivos intermedios?  
**A:** Sí—carga el shapefile en una `FeatureCollection` y llama directamente a `ExportToGeoJson()`. Consulta el tutorial “Extraer entidades a GeoJSON” para un ejemplo completo.

**Q:** ¿Necesito una licencia especial para ejecutar estos ejemplos localmente?  
**A:** Una licencia de evaluación gratuita funciona para desarrollo y pruebas. Para implementaciones en producción, adquiere una licencia comercial de Aspose.
