## Veelgestelde vragen

**Q: Kan ik deze conversies gebruiken in een productieomgeving?**  
A: Ja. Een commerciële Aspose.GIS‑licentie verwijdert alle proefbeperkingen en biedt volledige technische ondersteuning.

**Q: Welke .NET‑runtimes worden ondersteund?**  
A: De bibliotheek werkt met .NET Framework 4.6+, .NET Core 3.1+, .NET 5 en .NET 6.

**Q: Moet ik native GIS‑software installeren?**  
A: Nee. Aspose.GIS is een pure‑managed .NET‑bibliotheek; er zijn geen externe afhankelijkheden vereist.

**Q: Hoe groot een bestand kan ik converteren?**  
A: Bestanden tot enkele honderden megabytes worden moeiteloos verwerkt; voor zeer grote datasets kun je overwegen de streaming‑API’s van Aspose.GIS te gebruiken.

**Q: Is er een manier om coördinatenreferentiesystemen (CRS) te behouden tijdens conversie?**  
A: Ja. De API behoudt automatisch CRS‑informatie tenzij je de gegevens expliciet reprojectioneert.
