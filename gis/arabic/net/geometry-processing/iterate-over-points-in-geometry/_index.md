# كيفية إضافة نقاط وتكرارها على الشكل الهندسي

## المقدمة

إذا كنت تعمل مع بيانات GIS في بيئة .NET، فإن أحد أولى الأشياء التي تحتاج إلى معرفتها هو **كيفية إضافة نقاط** إلى شكل هندسي ثم العمل مع تلك النقاط. توفر Aspose.GIS for .NET واجهة برمجة تطبيقات كائنية نظيفة تجعل هذه العملية مباشرة. في هذا البرنامج التعليمي سنستعرض إنشاء `LineString`، إضافة نقاط إليه، وتكرار تلك النقاط حتى تتمكن من استخراج الإحداثيات أو إجراء تحليلات إضافية.

## إجابات سريعة
- **ما هو الصنف الأساسي لمجموعات النقاط؟** `LineString`
- **كيف تضيف نقطة؟** استخدم `AddPoint(longitude, latitude)`
- **هل يمكنك التكرار باستخدام حلقة foreach؟** نعم، `LineString` يطبق `IEnumerable<IPoint>`
- **المتطلبات المسبقة؟** .NET 6+ (أو .NET Core 3.1/Framework 4.6+) ومكتبة Aspose.GIS for .NET
- **حالة الاستخدام النموذجية؟** بناء المسارات، تصور المسارات، أو إعداد البيانات للتحليل المكاني

## ما معنى “إضافة نقاط” في GIS؟

إضافة النقاط تعني إدراج أزواج إحداثيات فردية (خط الطول، خط العرض) داخل حاوية هندسية مثل `LineString` أو `Polygon` أو `MultiPoint`. كل نقطة تصبح رأسًا يحدد الشكل أو المسار الذي تقوم بنمذجته.

## لماذا نضيف نقاط باستخدام Aspose.GIS؟

- **أمان نوع قوي** – كائنات الشكل الهندسي ذات نوعية قوية، مما يقلل الأخطاء أثناء التشغيل.  
- **متعدد المنصات** – يعمل على .NET Framework، .NET Core، و .NET 5/6+.  
- **واجهة برمجة تطبيقات غنية** – تتضمن التكرار المدمج، العمليات المكانية، ودعم الصيغ (Shapefile، GeoJSON، إلخ).

## المتطلبات المسبقة

- Visual Studio 2022 (أو أي بيئة تطوير C#)  
- حزمة NuGet الخاصة بـ Aspose.GIS for .NET مثبتة  
- فهم أساسي لصياغة C#  

## استيراد المساحات الاسمية

ابدأ باستيراد المساحات الاسمية الضرورية لتمكين الوصول إلى وظائف Aspose.GIS في تطبيق .NET الخاص بك:

```csharp
using Aspose.Gis.Geometries;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

## كيفية إضافة نقاط إلى شكل هندسي؟

### الخطوة 1: إنشاء كائن `LineString`  

يمثل `LineString` تسلسلًا من النقاط المتصلة (خط متعدد). أولاً، أنشئ الكائن:

```csharp
LineString line = new LineString();
```

### الخطوة 2: إضافة نقاط إلى `LineString`  

استخدم طريقة `AddPoint` لإدخال كل زوج إحداثيات. هذا هو جوهر **كيفية إضافة نقاط** إلى الشكل الهندسي الخاص بك:

```csharp
line.AddPoint(78.65, -32.65);
line.AddPoint(-98.65, 12.65);
```

يمكنك استدعاء `AddPoint` عددًا لا نهائيًا من المرات حسب الحاجة؛ كل استدعاء يضيف رأسًا جديدًا إلى الخط.

### الخطوة 3: التكرار على النقاط  

بعد إضافة النقاط، يمكنك التكرار عليها باستخدام جملة `foreach`. يطبق `LineString` الواجهة `IEnumerable<IPoint>`، مما يجعل التكرار بسيطًا وبديهيًا:

```csharp
foreach (IPoint point in line)
{
    Console.WriteLine(point.X + "," + point.Y);
}
```

تقوم الحلقة بطباعة قيم X (خط الطول) و Y (خط العرض) لكل نقطة إلى وحدة التحكم، مما يتيح لك التحقق من أن النقاط أضيفت بشكل صحيح.

## حالات الاستخدام الشائعة

- **تخطيط المسارات** – بناء مسار من سجلات GPS ثم تحليل المسافات بين نقاط الطريق.  
- **تحقق من صحة البيانات** – التكرار على النقاط للتأكد من أنها تقع ضمن الحدود المتوقعة (مثلاً داخل حدود دولة).  
- **التصور** – تصدير `LineString` إلى GeoJSON أو Shapefile لاستخدامه في أدوات الخرائط.
