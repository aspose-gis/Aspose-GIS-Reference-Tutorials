# كيفية حساب طول الهندسة في .NET باستخدام Aspose.GIS

## المقدمة
إذا كنت تبحث عن طريقة واضحة وعملية **كيفية حساب الطول** للكائنات الهندسية في تطبيق .NET، فقد وصلت إلى المكان المناسب. توفر لك Aspose.GIS for .NET مجموعة غنية من واجهات برمجة التطبيقات الموجهة لنظم المعلومات الجغرافية التي تجعل الحسابات المكانية—مثل قياس طول الخط أو محيط المضلع—سهلة وسريعة. في هذا الدرس سنستعرض العملية بالكامل، بدءًا من إعداد البيئة وصولاً إلى كتابة كود C# الذي يُعيد قيم طول دقيقة.

## إجابات سريعة
- **ماذا تُرجع الدالة “GetLength()”؟** تُرجع طول الخط للخطوط؛ وتُرجع المحيط للمضلعات.  
- **ما هو الـ namespace المطلوب؟** `Aspose.Gis.Geometries`.  
- **هل يمكنني استخدامها مع .NET 6؟** نعم، تدعم Aspose.GIS .NET 5، .NET 6، والإصدارات الأحدث.  
- **هل أحتاج إلى ترخيص للتطوير؟** النسخة التجريبية المجانية كافية للتقييم؛ يلزم الحصول على ترخيص للإنتاج.  
- **هل الحساب واعٍ للوحدات؟** يُرجع الطول بوحدات نظام الإحداثيات (مثلاً أمتار لنظام إحداثيات إسقاطية).

## ما هو طول الهندسة؟
`Geometry.GetLength()` هي طريقة تُرجع القياس الخطي لكائن الهندسة. بالنسبة لـ `LineString` تُعطي إجمالي طول الخط، بينما بالنسبة لـ `Polygon` تُرجع المحيط (مجموع جميع حوافه). تُبسط هذه الطريقة الرياضيات الأساسية، مما يتيح لك التركيز على منطق GIS عالي المستوى.

## لماذا نستخدم Aspose.GIS لحساب الطول؟
- **بدون تبعيات خارجية** – مكتبة .NET صافية، لا تحتاج إلى DLLs أصلية.  
- **دقة عالية** – تستخدم حسابات ذات دقة مزدوجة للحصول على نتائج دقيقة.  
- **متعددة المنصات** – تعمل على Windows وLinux وmacOS مع .NET Core/5/6+.  

## دليل خطوة بخطوة

### الخطوة 1: إنشاء كائنات الهندسة
لبدء العمل، أنشئ كائنات الهندسة التي تمثل الأشكال التي تريد حساب طولها. يمكن أن تشمل خطوطًا، مضلعات، أو أي أشكال هندسية أخرى.

```csharp
var line = new LineString();
line.AddPoint(0, 0);
line.AddPoint(2, 2);
line.AddPoint(2, 0);
```

### الخطوة 2: كيفية حساب طول الخط في C#
بعد إنشاء كائن الخط، يمكنك حساب طوله باستخدام طريقة `GetLength()`. هذا يُظهر **calculate line length c#** في سطر واحد من الكود.

```csharp
Console.WriteLine("{0:F}", line.GetLength()); // Output: 4.83
```

### الخطوة 3: إنشاء هندسة مضلع
بنفس الطريقة، يمكنك إنشاء كائنات هندسة المضلعات باستخدام الفئتين `Polygon` و `LinearRing`.

```csharp
var rectangle = new Polygon(new LinearRing(new[]
{
    new Point(0, 0),
    new Point(0, 1),
    new Point(1, 1),
    new Point(1, 0),
    new Point(0, 0),
}));
```

### الخطوة 4: كيفية الحصول على طول المضلع
بالنسبة للمضلعات، تُرجع طريقة `GetLength()` المحيط، وهو ما يُعادل **how to get length** للشكل.

```csharp
Console.WriteLine("{0:F}", rectangle.GetLength()); // Output: 4.00
```
