# كيفية استخدام Aspose: حساب الغلاف المحدب باستخدام Aspose.GIS لـ .NET

## مقدمة
في هذا الدرس، **ستتعلم كيفية حساب الغلاف المحدب** لشكل هندسي في تطبيق .NET باستخدام Aspose.GIS. سواءً كنت تبني أداة رسم خرائط، أو تُجري تحليلًا مكانيًا، أو تحتاج ببساطة إلى تحديد حدود مجموعة من النقاط، فإن عملية الغلاف المحدب تُعدّ حجر الأساس الأساسي. سنستعرض كل شيء—من إعداد المشروع إلى استخراج نقاط الغلاف المحدب—حتى تتمكن من دمج هذه القدرة بثقة.

## إجابات سريعة
- **ماذا يعني “الغلاف المحدب”؟** هو أصغر مضلع محدب يحيط بالكامل بمجموعة من النقاط.  
- **أي مكتبة توفر حساب الغلاف؟** Aspose.GIS لـ .NET تقدم طريقة مدمجة `GetConvexHull()`.  
- **هل أحتاج إلى ترخيص لتشغيل العينة؟** النسخة التجريبية المجانية تكفي للتقييم؛ الترخيص التجاري مطلوب للإنتاج.  
- **ما إصدارات .NET المدعومة؟** .NET Framework 4.5+، .NET Core 3.1+، .NET 5/6/7.  
- **هل يمكن استخراج نقاط الغلاف الفردية؟** نعم—قم بتحويل النتيجة إلى `ILinearRing` وتكرار إحداثياتها.

## لماذا نحسب الغلاف المحدب باستخدام Aspose.GIS؟
- **أداء عالي** – خوارزميات أصلية مُحسّنة تتعامل مع آلاف النقاط فورًا.  
- **لا تبعيات خارجية** – لا حاجة لمحركات هندسية طرف ثالث.  
- **دعم صيغ غني** – يعمل مع ملفات shapefile، GeoJSON، KML، وأكثر، بحيث يمكنك تغذية أي مصدر بيانات إلى حساب الغلاف.  
- **واجهة برمجة تطبيقات متسقة** – النمط السلس نفسه الذي تستخدمه للعمليات المكانية الأخرى يحافظ على نظافة وصيانة الكود.

## كيفية حساب الغلاف المحدب باستخدام Aspose.GIS لـ .NET
### الخطوة 1: إنشاء شكل هندسي MultiPoint
أولاً، عرّف شكلًا هندسيًا متعدد النقاط يحتوي على عدة نقاط. ستشكل هذه النقاط الأساس لحساب الغلاف المحدب.

```csharp
var geometry = new MultiPoint
{
    new Point(3, 2),
    new Point(0, 0),
    new Point(6, 5),
    new Point(5, 10),
    new Point(10, 0),
    new Point(8, 2),
    new Point(4, 3),
};
```
يُنشئ هذا المقتطف كودًا شكلًا هندسيًا متعدد النقاط يضم سبع نقاط متميزة.

### الخطوة 2: الحصول على الغلاف المحدب
بعد ذلك، استدعِ طريقة `GetConvexHull()` على كائن الشكل الهندسي لحساب الغلاف المحدب.

```csharp
var convexHull = geometry.GetConvexHull();
```
تحسب هذه الطريقة الغلاف المحدب للشكل الهندسي المدخل، وتنتج شكلًا هندسيًا جديدًا يمثل الغلاف المحدب.

### الخطوة 3: الوصول إلى نقاط الغلاف المحدب
بمجرد حساب الغلاف المحدب، يمكنك **استخراج نقاط الغلاف المحدب** عن طريق تحويل النتيجة إلى `ILinearRing` وتكرار رؤوسه.

```csharp
var ring = (ILinearRing)convexHull;
for (int i = 0; i < ring.Count; ++i)
{
    Console.WriteLine("[{0}] = ({1} {2})", i, ring[i].X, ring[i].Y);
}
```
يتكرر هذا الحلقة عبر نقاط الغلاف المحدب ويطبع إحداثياتها إلى وحدة التحكم.
