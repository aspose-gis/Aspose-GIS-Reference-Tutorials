## 常见问题

**Q:** 我如何以编程方式 **创建新 shapefile** 文件？  
**A:** 使用 Aspose.GIS 中的 `Shapefile` 类，指定几何类型，添加字段，然后写入要素。链接的 “Create New Shapefile” 教程展示了完整代码。

**Q:** File GDB 与 shapefile 有何区别？  
**A:** File Geodatabase（File GDB）以文件夹结构存储数据，支持更丰富的模式；而 shapefile 是一种简单、兼容性广的格式，由多个固定长度文件组成。

**Q:** 我如何高效地 **按属性过滤要素**？  
**A:** 构建带有 WHERE 子句的 `Query` 对象（例如 `SELECT * FROM layer WHERE Population > 1000`）。 “Filter Features by Attribute” 指南会带您逐步了解 API。

**Q:** 我能在不使用中间文件的情况下将 shapefile 转换为 GeoJSON 吗？  
**A:** 可以——将 shapefile 加载到 `FeatureCollection`，直接调用 `ExportToGeoJson()`。请参阅 “Extract Features to GeoJSON” 教程获取完整示例。

**Q:** 本地运行这些示例是否需要特殊许可证？  
**A:** 免费评估许可证可用于开发和测试。生产部署需购买 Aspose 的商业许可证。
