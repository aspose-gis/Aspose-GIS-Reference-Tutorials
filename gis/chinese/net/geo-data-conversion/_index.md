## 常见问题

**Q: 我可以在生产环境中使用这些转换吗？**  
A: 是的。商业版 Aspose.GIS 许可证消除所有试用限制，并提供完整的技术支持。

**Q: 支持哪些 .NET 运行时？**  
A: 该库兼容 .NET Framework 4.6+、.NET Core 3.1+、.NET 5 和 .NET 6。

**Q: 我需要安装任何本地 GIS 软件吗？**  
A: 不需要。Aspose.GIS 是纯托管的 .NET 库，无需外部依赖。

**Q: 我能转换多大的文件？**  
A: 可轻松处理数百兆字节的文件；对于超大数据集，请考虑 Aspose.GIS 提供的流式 API。

**Q: 转换过程中是否可以保留坐标参考系统 (CRS)？**  
A: 可以。除非您显式重新投影数据，API 会自动保留 CRS 信息。
