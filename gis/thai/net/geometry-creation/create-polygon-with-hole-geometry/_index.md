# สร้างรูปหลายเหลี่ยมที่มีรูภายในโดยใช้ Aspose.GIS

## บทนำ
ในบทแนะนำนี้ คุณจะ **สร้างรูปหลายเหลี่ยมที่มีรูภายใน** โดยใช้ Aspose.GIS สำหรับ .NET ไม่ว่าคุณจะกำลังสร้างแอปพลิเคชันแผนที่, ทำการวิเคราะห์เชิงพื้นที่, หรือเตรียมข้อมูลสำหรับบริการ GIS การรู้วิธีฝังรูภายในรูปหลายเหลี่ยมเป็นสิ่งสำคัญ เราจะเดินผ่านกระบวนการทั้งหมดทีละขั้นตอน ตั้งแต่การตั้งค่าสภาพแวดล้อมจนถึงการสร้างอ็อบเจกต์รูปหลายเหลี่ยมขั้นสุดท้าย

## คำตอบเร็ว
- **What does “create polygon with hole” mean?** หมายความว่าการสร้างรูปหลายเหลี่ยมที่มีหนึ่งหรือหลายวงแหวนภายใน (รู) ซึ่งจะไม่รวมอยู่ในพื้นที่
- **Which library handles this?** Aspose.GIS for .NET ให้การสนับสนุนเต็มรูปแบบสำหรับวงแหวนภายนอกและภายใน
- **Do I need a license?** รุ่นทดลองฟรีใช้ได้สำหรับการพัฒนา; จำเป็นต้องมีลิขสิทธิ์เชิงพาณิชย์สำหรับการใช้งานจริง
- **What .NET versions are supported?** รองรับ .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7
- **How long does it take?** โดยทั่วไปใช้เวลาน้อยกว่า 10 นาทีในการทำและทดสอบ

## “สร้างรูปหลายเหลี่ยมที่มีรูภายใน” คืออะไร?
การสร้างรูปหลายเหลี่ยมที่มีรูภายในหมายถึงการกำหนด **วงแหวนภายนอก** ที่รอบขอบนอกของรูปและหนึ่งหรือหลาย **วงแหวนภายใน** ที่ตัดพื้นที่ว่างออก วงแหวนภายในมักเรียกว่า *รู* เนื่องจากเป็นพื้นที่ที่ไม่ได้เป็นส่วนหนึ่งของพื้นผิวรูปหลายเหลี่ยม

## ทำไมต้องสร้างรูในรูปหลายเหลี่ยมโดยใช้ Aspose.GIS?

## นำเข้า Namespaces
ก่อนอื่น คุณต้องนำเข้า namespaces ที่จำเป็นเพื่อทำงานกับฟังก์ชันของ Aspose.GIS ดังนี้:

```csharp
using Aspose.Gis.Geometries;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

ตอนนี้เราจะดำเนินการสร้างรูปหลายเหลี่ยมที่มีรูภายในโดยใช้ Aspose.GIS สำหรับ .NET

## ขั้นตอนที่ 1: สร้างอ็อบเจกต์ Polygon
เราจะเริ่มโดยการสร้างอ็อบเจกต์ `Polygon` ว่างที่ภายหลังจะเก็บวงแหวนภายนอกและวงแหวนภายใน

```csharp
Polygon polygon = new Polygon();
```

## ขั้นตอนที่ 2: กำหนดวงแหวนภายนอก
วงแหวนภายนอกกำหนดขอบนอกของรูปหลายเหลี่ยม เพิ่มจุดในลำดับตามเข็มนาฬิกาเพื่อสร้างรูปปิด

```csharp
LinearRing ring = new LinearRing();
ring.AddPoint(50.02, 36.22);
ring.AddPoint(49.99, 36.26);
ring.AddPoint(49.97, 36.23);
ring.AddPoint(49.98, 36.17);
ring.AddPoint(50.02, 36.22);
```

## ขั้นตอนที่ 3: กำหนดวงแหวนภายใน (รู)
ต่อไป เราจะสร้างวงแหวนภายใน — นี่คือ **รู** ที่จะถูกตัดออกจากพื้นที่ของรูปหลายเหลี่ยม จุดมักจะเพิ่มในลำดับทวนเข็มนาฬิกา แต่ Aspose.GIS จะจัดการการกำหนดทิศทางโดยอัตโนมัติ

```csharp
LinearRing hole = new LinearRing();
hole.AddPoint(50.00, 36.22);
hole.AddPoint(49.99, 36.20);
hole.AddPoint(49.98, 36.23);
hole.AddPoint(50.00, 36.24);
hole.AddPoint(50.00, 36.22);
```

## ขั้นตอนที่ 4: กำหนดวงแหวนภายนอกและเพิ่มวงแหวนภายในลงใน Polygon
สุดท้าย เราแนบวงแหวนภายนอกเข้ากับรูปหลายเหลี่ยมแล้วเพิ่มวงแหวนภายใน (รู) วิธี `AddInteriorRing` สามารถเรียกหลายครั้งหากต้องการหลายรู

```csharp
polygon.ExteriorRing = ring;
polygon.AddInteriorRing(hole);
```

## ปัญหาที่พบบ่อยและวิธีแก้
| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|--------|-----|
| รูไม่แสดงใน GIS viewer | การกำหนดทิศทางของวงแหวนภายในกลับกัน | ตรวจสอบให้จุดถูกเพิ่มในทิศทางตรงข้ามกับวงแหวนภายนอก (ทวนเข็มนาฬิกา) |
| ข้อผิดพลาดรูปหลายเหลี่ยมไม่ถูกต้อง | วงแหวนไม่ปิด (จุดแรก ≠ จุดสุดท้าย) | ทำซ้ำจุดแรกเป็นจุดสุดท้ายของแต่ละวงแหวน (ตามที่แสดงข้างต้น) |
| รูปทรงว่างโดยไม่คาดคิด | ลืมกำหนด `ExteriorRing` ก่อนเพิ่มวงแหวนภายใน | ตั้งค่า `polygon.ExteriorRing` ก่อน, จากนั้นเรียก `AddInteriorRing` |
