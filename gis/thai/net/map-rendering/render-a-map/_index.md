# วิธีสร้างแผนที่ SVG และเพิ่มเมืองด้วย Aspose.GIS สำหรับ .NET

## บทนำ

ยินดีต้อนรับสู่โลกที่น่าตื่นเต้นของ Aspose.GIS สำหรับ .NET! ในคู่มือขั้นตอนนี้ คุณจะได้เรียนรู้ **วิธีเพิ่มเมืองลงในแผนที่** และ **สร้างแผนที่ SVG** ที่ดูสวยงามบนหน้าจอใดก็ได้ ไม่ว่าคุณจะสร้างแดชบอร์ดบนเดสก์ท็อป, พอร์ทัล GIS บนเว็บ, หรือรายงานที่พิมพ์ได้ โค้ดเดียวกันทำงานได้บน .NET Framework, .NET Core, และ .NET 5/6 มาดำเนินการทั้งหมดตั้งแต่การกำหนดขนาดแผนที่จนถึงการส่งออกไฟล์ SVG ที่สะอาดและปรับขนาดได้

## คำตอบสั้น
- **สิ่งที่บทเรียนนี้ครอบคลุมคืออะไร?** การเพิ่มจุดเมืองลงในแผนที่และส่งออกผลลัพธ์เป็นไฟล์ SVG.  
- **ไลบรารีที่ต้องการคืออะไร?** Aspose.GIS for .NET (มีรุ่นทดลองฟรี).  
- **ฉันต้องการใบอนุญาตหรือไม่?** รุ่นทดลองใช้ได้สำหรับการพัฒนา; ต้องมีใบอนุญาตเชิงพาณิชย์สำหรับการใช้งานในผลิตภัณฑ์.  
- **ฉันสามารถใช้ในแอปเว็บได้หรือไม่?** แน่นอน – โค้ดเดียวกันทำงานใน ASP.NET, Blazor, และเฟรมเวิร์กเว็บ .NET อื่น ๆ.  
- **รูปแบบผลลัพธ์ที่สร้างคืออะไร?** แผนที่ SVG ความละเอียดสูงที่เบราว์เซอร์แสดงผลทันทีและโปรแกรมแก้ไขเวกเตอร์สามารถแก้ไขต่อได้.

## “generate SVG map” คืออะไร?
*Generating an SVG map* หมายถึงการแปลงข้อมูลทางภูมิศาสตร์เป็นไฟล์ Scalable Vector Graphics ที่คงรูปทรง, สไตล์, และความโต้ตอบโดยไม่ทำให้ภาพเป็น raster. ไฟล์ SVG จะคมชัดที่ระดับการซูมใด ๆ และเหมาะสำหรับการแสดงผลบนเว็บ.

## ทำไมต้องสร้างแผนที่ SVG ด้วย Aspose.GIS?
Aspose.GIS รองรับ **รูปแบบข้อมูลเข้าและออกกว่า 70 แบบ** (รวมถึง Shapefile, GeoJSON, KML, และ GML) และสามารถเรนเดอร์แผนที่ด้วย **ความแม่นยำระดับ sub‑pixel** พร้อมการใช้หน่วยความจำน้อย. ไลบรารีนี้ประมวลผล **ชุดข้อมูลหลายร้อยหน้า** โดยไม่ต้องโหลดไฟล์ทั้งหมดเข้าสู่หน่วยความจำ ทำให้เหมาะกับโครงการ GIS ขนาดใหญ่.

## ข้อกำหนดเบื้องต้น
ก่อนเริ่มทำตามบทเรียน โปรดตรวจสอบว่าคุณมีข้อกำหนดต่อไปนี้พร้อมแล้ว:
- Aspose.GIS for .NET Library: ตรวจสอบว่าคุณได้ติดตั้งไลบรารี Aspose.GIS for .NET แล้ว คุณสามารถดาวน์โหลดได้จาก [ที่นี่](https://releases.aspose.com/gis/net/).
- Data Files: เตรียมไฟล์ shapefile และข้อมูล GeoJSON ที่จำเป็นสำหรับบทเรียน คุณสามารถค้นหาข้อมูลตัวอย่างในเอกสารหรือใช้ไฟล์ของคุณเอง.
- Development Environment: ตั้งค่าสภาพแวดล้อมการพัฒนา .NET รวมถึงโปรแกรมแก้ไขโค้ดเช่น Visual Studio.

## นำเข้า Namespace
Namespace `Aspose.Gis` ให้ฟังก์ชัน GIS หลักทั้งหมด ในขณะที่ `Aspose.Gis.Rendering` มีคลาสที่เกี่ยวกับการเรนเดอร์.

```csharp
using Aspose.Gis;
using Aspose.Gis.Rendering;
using Aspose.Gis.Rendering.Symbolizers;
using Aspose.Gis.SpatialReferencing;
using Aspose.GIS.Examples.CSharp;
using System.Drawing;
using System.Drawing.Drawing2D;
using System.Drawing.Text;
using System.IO;
using System.Linq;
```

## วิธีตั้งค่าขนาดแผนที่
`Map` คือคลาสหลักที่เก็บพื้นผิวการเรนเดอร์และจัดการเลเยอร์.  
ตั้งค่าขนาดแคนวาสของแผนที่ก่อนเพื่อให้เรนเดอร์รู้ว่าจะจัดสรรพื้นที่เท่าไหร่ คุณสร้างอ็อบเจ็กต์ `Map` ส่งค่าความกว้างและความสูงเป็นพิกเซล และอาจกำหนดสีพื้นหลังได้ ขั้นตอนนี้ควบคุมอัตราส่วนภาพของ SVG, ขนาดไฟล์, และความคมชัดบนอุปกรณ์ต่าง ๆ.

```csharp
string dataDir = "Your Document Directory";
using (var map = new Map(800, 476))
{
    // Additional code for map setup can be added here.
}
```

## วิธีวาดเลเยอร์เวกเตอร์สำหรับแผนที่ฐาน
`DrawVectorLayer` ทำการเรนเดอร์เลเยอร์เวกเตอร์บนแผนที่โดยใช้ symbolizer ที่กำหนด.  
คลาส `Map` มีเมธอด `DrawVectorLayer` ที่อ่านไฟล์ shapefile และเรนเดอร์แต่ละฟีเจอร์ด้วยสไตล์ `SimpleFill` คุณสามารถปรับสีเติม, ความหนาของเส้นขอบ, และความทึบเพื่อให้ตรงกับธีมภาพของคุณ และยังสามารถใช้ความโปร่งแสงหรือเติมลวดลายเพื่อเพิ่มเอฟเฟกต์การทำแผนที่ที่หลากหลาย.

```csharp
var baseMapSymbolizer = new SimpleFill { FillColor = Color.Salmon, StrokeWidth = 0.75 };
map.Add(VectorLayer.Open(dataDir + "basemap.shp", Drivers.Shapefile), baseMapSymbolizer);
```

## วิธีโหลด GeoJSON และเพิ่มเมืองลงในแผนที่
`FeatureBasedConfiguration` คือ delegate ที่ให้คุณกำหนดสไตล์ให้แต่ละฟีเจอร์ตามแอตทริบิวต์ของมัน.  
การโหลดไฟล์ GeoJSON จะให้คอลเลกชันของฟีเจอร์จุดที่แทนเมืองต่าง ๆ โดยการส่งผ่าน delegate `FeatureBasedConfiguration` คุณสามารถกำหนดสไตล์ให้แต่ละเครื่องหมายเมืองตามแอตทริบิวต์เช่นประชากรหรือภูมิภาค คุณยังสามารถกรองฟีเจอร์หรือปรับขนาดสัญลักษณ์แบบไดนามิกเพื่อสะท้อนมิติข้อมูลเพิ่มเติมได้.

```csharp
var citiesSymbolizer = new SimpleMarker() { FillColor = Color.LightBlue };
citiesSymbolizer.FeatureBasedConfiguration = (feature, symbolizer) =>
{
    // Additional configuration logic can be added here.
};
map.Add(VectorLayer.Open(dataDir + "points.geojson", Drivers.GeoJson), citiesSymbolizer);
```

## วิธีส่งออกแผนที่เป็นไฟล์ SVG
`Map.Save` ส่งออกแผนที่ที่เรนเดอร์เป็นไฟล์ในรูปแบบที่ระบุ.  
การเรียก `Map.Save` พร้อมอาร์กิวเมนต์ `SaveFormat.Svg` จะเขียนข้อมูลเวกเตอร์ที่เรนเดอร์ลงในไฟล์ SVG บนดิสก์ ไฟล์ `cities_out.svg` ที่ได้สามารถเปิดโดยตรงในเบราว์เซอร์สมัยใหม่หรือแก้ไขด้วยเครื่องมือเช่น Inkscape คุณยังสามารถกำหนด DPI หรือฝัง CSS เพื่อสไตล์เพิ่มเติมได้.

```csharp
map.Render(dataDir + "cities_out.svg", Renderers.Svg);
```

## ปัญหาทั่วไปและวิธีแก้
- **Missing data file error** – ตรวจสอบว่าเส้นทางสัมพัทธ์ไปยัง shapefile และ GeoJSON ของคุณถูกต้อง; ใช้เส้นทางเต็มระหว่างการดีบัก.  
- **Cities not visible** – ตรวจสอบให้แน่ใจว่าระบบอ้างอิงพิกัดของ GeoJSON ตรงกับ CRS ของแผนที่ฐาน, หรือทำการรีโปรเจกต์ข้อมูลโดยใช้ `Geometry.Transform`.  
- **SVG appears blank** – ตรวจสอบว่าสีพื้นหลังของแผนที่ไม่ได้ตั้งเป็นโปร่งใสเต็ม; ตั้งสีเติมอ่อนเพื่อยืนยันการเรนเดอร์.
