# วิธีใช้ Aspose: คำนวณ Convex Hull ด้วย Aspose.GIS สำหรับ .NET

## Introduction
ในบทแนะนำนี้, **คุณจะได้เรียนรู้วิธีคำนวณ convex hull** ของรูปทรงเรขาคณิตในแอปพลิเคชัน .NET ด้วย Aspose.GIS ไม่ว่าคุณจะกำลังสร้างเครื่องมือแผนที่, ทำการวิเคราะห์เชิงพื้นที่, หรือเพียงต้องการร่างขอบเขตของชุดจุด, การดำเนินการ convex hull เป็นบล็อกพื้นฐานที่สำคัญ เราจะพาคุณผ่านทุกขั้นตอน—ตั้งแต่การตั้งค่าโครงการจนถึงการดึงจุดของ convex hull—เพื่อให้คุณสามารถรวมความสามารถนี้เข้าไปในโค้ดของคุณได้อย่างมั่นใจ

## Quick Answers
- **“convex hull” หมายถึงอะไร?** เป็นรูปหลายเหลี่ยมคอนเว็กซ์ที่เล็กที่สุดซึ่งล้อมรอบชุดจุดทั้งหมดอย่างสมบูรณ์.  
- **ไลบรารีใดให้การคำนวณ hull?** Aspose.GIS สำหรับ .NET มีเมธอด `GetConvexHull()` ในตัว.  
- **ต้องมีลิขสิทธิ์เพื่อรันตัวอย่างหรือไม่?** สามารถใช้รุ่นทดลองฟรีเพื่อประเมิน; ต้องมีลิขสิทธิ์เชิงพาณิชย์สำหรับการใช้งานจริง.  
- **รองรับเวอร์ชัน .NET ใดบ้าง?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **สามารถดึงจุด hull แต่ละจุดได้หรือไม่?** ได้—แคสผลลัพธ์เป็น `ILinearRing` แล้ววนลูปพิกัดของมัน.

## How to calculate convex hull with Aspose.GIS for .NET
### Step 1: Create a MultiPoint Geometry
กำหนดรูปทรง MultiPoint ที่ประกอบด้วยหลายจุด จุดเหล่านี้จะเป็นฐานสำหรับการคำนวณ convex hull

```csharp
var geometry = new MultiPoint
{
    new Point(3, 2),
    new Point(0, 0),
    new Point(6, 5),
    new Point(5, 10),
    new Point(10, 0),
    new Point(8, 2),
    new Point(4, 3),
};
```
โค้ดสแนปนี้สร้างรูปทรง MultiPoint ที่มีจุดเจ็ดจุดที่แตกต่างกัน

### Step 2: Get Convex Hull
เรียกเมธอด `GetConvexHull()` บนอ็อบเจ็กต์รูปทรงเพื่อคำนวณ convex hull

```csharp
var convexHull = geometry.GetConvexHull();
```
เมธอดนี้คำนวณ convex hull ของรูปทรงอินพุตและสร้างรูปทรงใหม่ที่เป็นตัวแทนของ convex hull

### Step 3: Access Convex Hull Points
เมื่อได้ convex hull แล้ว คุณสามารถ **ดึงจุด convex hull** โดยแคสผลลัพธ์เป็น `ILinearRing` แล้ววนลูปผ่านเวอร์เท็กซ์ของมัน

```csharp
var ring = (ILinearRing)convexHull;
for (int i = 0; i < ring.Count; ++i)
{
    Console.WriteLine("[{0}] = ({1} {2})", i, ring[i].X, ring[i].Y);
}
```
ลูปนี้วนผ่านจุดของ convex hull และพิมพ์พิกัดของแต่ละจุดลงคอนโซล

## Common Issues and Solutions
- **ผลลัพธ์เป็น null:** ตรวจสอบให้แน่ใจว่ารูปทรงต้นทางมีจุดอย่างน้อยสามจุดที่ไม่อยู่บนเส้นตรงเดียวกัน; มิฉะนั้น `GetConvexHull()` อาจคืนรูปทรงเดิม.  
- **การแคสที่ไม่ถูกต้อง:** hull ถูกคืนเป็นอ็อบเจ็กต์ `Geometry`; การแคสเป็น `ILinearRing` ปลอดภัยเฉพาะเมื่อผลลัพธ์เป็นวงแหวนรูปหลายเหลี่ยม ตรวจสอบประเภทก่อนแคสหากทำงานกับคอลเลกชันรูปทรงผสม.  
- **ข้อยกเว้นลิขสิทธิ์:** การรันโค้ดโดยไม่มีลิขสิทธิ์ที่ถูกต้องจะฝังลายน้ำในไฟล์ที่สร้างขึ้น; ควรรับลิขสิทธิ์ทดลองหรือเชิงพาณิชย์เพื่อหลีกเลี่ยง.
