# ตรวจสอบจุดภายในโพลิกอนและรับจุดบนพื้นผิว

## บทนำ
ในบทแนะนำนี้คุณจะได้เรียนรู้ **วิธีตรวจสอบจุดภายในโพลิกอน** ด้วย Aspose.GIS สำหรับ .NET และยังได้เห็นวิธี **รับจุดบนพื้นผิว** ของรูปทรงเรขาคณิต เราจะอธิบายขั้นตอนการสร้างรูปทรงโพลิกอนใน C# การดึงจุดที่อยู่บนพื้นผิวของโพลิกอน และการตรวจสอบว่าจุดนั้นอยู่จริงภายในโพลิกอน ในตอนท้ายคุณจะได้โค้ดสคริปต์พร้อมใช้งานที่สามารถนำไปใส่ในแอปพลิเคชันภูมิสารสนเทศ .NET ใดก็ได้

## คำตอบสั้น
- **“check point inside polygon” หมายถึงอะไร?** มันตรวจสอบว่าพิกัดที่กำหนดอยู่ภายในขอบเขตของรูปทรงโพลิกอนหรือไม่  
- **เมธอดใดที่คืนค่าจุดภายในโพลิกอน?** `GetPointOnSurface()` คืนค่าจุดที่รับประกันว่าจะอยู่ภายในโพลิกอน  
- **ฉันต้องมีไลเซนส์เพื่อรันตัวอย่างหรือไม่?** รุ่นทดลองฟรีใช้ได้สำหรับการประเมิน; ต้องมีไลเซนส์เต็มเพื่อการใช้งานจริง  
- **เวอร์ชัน .NET ใดที่รองรับ?** .NET Framework, .NET Core, และ .NET Standard ทั้งหมดเข้ากันได้  
- **การทำงานนี้ใช้เวลานานเท่าไหร่?** ประมาณ 5‑10 นาทีสำหรับการคัดลอก, คอมไพล์, และรัน  

## “check point inside polygon” คืออะไร?
การตรวจสอบจุดภายในโพลิกอนเป็นการดำเนินการเชิงพื้นที่พื้นฐาน มันตอบคำถามว่า “พิกัดนี้อยู่ในพื้นที่ที่กำหนดโดยโพลิกอนหรือไม่?” ซึ่งเป็นสิ่งสำคัญสำหรับงานเช่น การกำหนดเขตพื้นที่ (geofencing), การวิเคราะห์แผนที่, และการตรวจสอบเชิงพื้นที่  

## ทำไมต้องใช้ Aspose.GIS สำหรับงานนี้?
Aspose.GIS ให้ API ที่มีประสิทธิภาพสูงและจัดการได้เต็มรูปแบบซึ่งจัดการการดำเนินการรูปทรงเรขาคณิตที่ซับซ้อนได้โดยไม่ต้องพึ่งพาไลบรารีภายนอก รองรับระบบอ้างอิงพิกัดหลากหลาย ทำงานบน .NET runtime หลักทั้งหมด และมีเมธอดที่ชัดเจนและเชื่อมต่อกันได้ เช่น `SpatiallyContains()` และ `GetPointOnSurface()`

## คู่มือขั้นตอนต่อขั้นตอน

### ขั้นตอนที่ 1: สร้างรูปทรงโพลิกอนใน C#
แรก เราต้อง **สร้างโพลิกอน** รูปทรงเรขาคณิต เรากำหนดวงแหวนภายนอกของโพลิกอนโดยระบุจุดยอดของมัน.

```csharp
var polygon = new Polygon();
polygon.ExteriorRing = new LinearRing(new[]
{
    new Point(0, 0),
    new Point(0, 1),
    new Point(1, 1),
    new Point(0, 0),
});
```

### ขั้นตอนที่ 2: รับจุดบนพื้นผิว
ต่อไป เราจะดึงจุดบนพื้นผิวของโพลิกอนโดยใช้เมธอด `GetPointOnSurface()` นี่คือขั้นตอน **รับจุดบนพื้นผิว**.

```csharp
IPoint pointOnSurface = polygon.GetPointOnSurface();
```

### ขั้นตอนที่ 3: ตรวจสอบจุดภายในโพลิกอน
เราสามารถตรวจสอบว่าจุดที่ดึงมานั้นอยู่ภายในโพลิกอนหรือไม่โดยใช้เมธอด `SpatiallyContains()` ซึ่งแสดงให้เห็นการ **ดึงจุดบนโพลิกอน** แล้วตรวจสอบมัน.

```csharp
Console.WriteLine(polygon.SpatiallyContains(pointOnSurface)); // True
```

## ปัญหาที่พบบ่อยและวิธีแก้
- **โพลิกอนว่าง** – ตรวจสอบให้วงแหวนภายนอกมีจุดยอดที่แตกต่างอย่างน้อยสามจุด; มิฉะนั้น `GetPointOnSurface()` อาจคืนค่าจุดที่ไม่ได้กำหนด  
- **ตามเข็มนาฬิกา vs. ทวนเข็มนาฬิกา** – การจัดเรียงของวงแหวนไม่ส่งผลต่อการตรวจสอบการครอบคลุม แต่การรักษาลำดับการหมุนที่สม่ำเสมอช่วยในงานเชิงพื้นที่อื่น ๆ  
- **ระบบพิกัด** – ตัวอย่างนี้ใช้ระนาบคาร์ทีเซียนแบบง่าย; เมื่อทำงานกับพิกัดจริง ควรตรวจสอบให้ระบบ CRS (coordinate reference system) ถูกกำหนดอย่างถูกต้อง
