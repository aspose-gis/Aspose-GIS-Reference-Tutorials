# ตรวจสอบ Geometry ครอบอีกอันหนึ่ง

## บทนำ
Aspose.GIS for .NET เป็นไลบรารีที่ทรงพลังซึ่งมอบเครื่องมือให้กับนักพัฒนาในการทำงานกับข้อมูลทางภูมิศาสตร์อย่างมีประสิทธิภาพภายในแอปพลิเคชัน .NET ของคุณ ไม่ว่าคุณจะกำลังสร้างแอปแผนที่, วิเคราะห์ข้อมูลเชิงพื้นที่, หรือรวมฟีเจอร์ทางภูมิศาสตร์เข้าไปในซอฟต์แวร์ของคุณ, Aspose.GIS มีชุดฟังก์ชันครบถ้วนเพื่อทำให้กระบวนการพัฒนาของคุณเป็นเรื่องง่าย ในบทเรียนนี้คุณจะได้เรียนรู้ **วิธีสร้าง LineString ใน C#**, เพิ่มจุดลงในเส้น, และทำ **การตรวจสอบจุดบนเส้น** ด้วยเมธอด `Covers` และ `CoveredBy`.

## คำตอบสั้น
- **“สร้าง LineString ใน C#” หมายถึงอะไร?** หมายถึงการสร้างอ็อบเจ็กต์ `LineString` แล้วเติมจุดพิกัดลงไป  
- **เมธอดใดใช้ตรวจสอบว่าจุดอยู่บนเส้น?** ใช้เมธอด `Covers` บน `LineString` หรือ `CoveredBy` บน `Point`  
- **ต้องมีลิขสิทธิ์เพื่อรันตัวอย่างหรือไม่?** ลิขสิทธิ์ชั่วคราวใช้ได้สำหรับการประเมิน; ต้องมีลิขสิทธิ์เต็มสำหรับการใช้งานจริง  
- **สามารถใช้กับ .NET Core ได้หรือไม่?** ใช่, Aspose.GIS รองรับ .NET Framework และ .NET Core  
- **สามารถเพิ่มจุดลงใน LineString ได้กี่จุด?** ไม่มีขีดจำกัดที่แน่นอน; คุณสามารถเพิ่มได้ตามต้องการสำหรับการวิเคราะห์เชิงพื้นที่ของคุณ  

## สิ่งที่หมายถึง **create LineString C#**?
`LineString` คือรูปทรงเรขาคณิตที่ประกอบด้วยรายการจุดที่เรียงลำดับกันโดยเส้นตรงเชื่อมต่อกัน ใน C# คุณสร้างมันโดยการสร้างอินสแตนซ์ของคลาส `LineString` จากเนมสเปซ `Aspose.Gis.Geometries` แล้ว **เพิ่มจุดลงใน LineString** ด้วยเมธอด `AddPoint`.

## นำเข้า Namespaces
ก่อนใช้ Aspose.GIS for .NET ในโปรเจกต์ของคุณ, คุณต้องนำเข้า Namespaces ที่จำเป็น:

using System;

## วิธี **create LineString C#** – คู่มือขั้นตอนโดยละเอียด

### ขั้นตอนที่ 1: สร้างอ็อบเจ็กต์ LineString
```csharp
var line = new LineString();
```
ที่นี่ เราสร้างอินสแตนซ์ใหม่ของอ็อบเจ็กต์ `LineString` ซึ่งแทนลำดับของเส้นเชื่อมต่อกันในพื้นที่สองมิติ

### ขั้นตอนที่ 2: **เพิ่มจุดลงใน LineString**
```csharp
line.AddPoint(0, 0);
line.AddPoint(1, 1);
```
เราจะ **เพิ่มจุดลงใน LineString** ด้วยเมธอด `AddPoint` ตัวอย่างนี้เพิ่มจุดสองจุด: (0, 0) และ (1, 1) ซึ่งสร้างเส้นทแยงมุมง่าย ๆ

### ขั้นตอนที่ 3: สร้างอ็อบเจ็กต์ Point
```csharp
var point = new Point(0, 0);
```
สร้างอ็อบเจ็กต์ `Point` ที่แทนจุดเดียวในพื้นที่สองมิติ ที่นี่เราสร้างจุดที่พิกัด (0, 0)

### ขั้นตอนที่ 4: ทำการ **ตรวจสอบจุดบนเส้น** – เส้นครอบจุดหรือไม่?
```csharp
Console.WriteLine(line.Covers(point));    // True
```
ใช้เมธอด `Covers` เพื่อตรวจสอบว่าเส้นครอบจุดหรือไม่ ในกรณีนี้จะคืนค่า `True` เนื่องจากจุด (0, 0) อยู่บนเส้นอย่างแม่นยำ

### ขั้นตอนที่ 5: ตรวจสอบความสัมพันธ์ย้อนกลับ – จุดถูกเส้นครอบหรือไม่?
```csharp
Console.WriteLine(point.CoveredBy(line)); // True
```
เช่นเดียวกัน ใช้เมธอด `CoveredBy` เพื่อตรวจสอบว่าจุดถูกเส้นครอบหรือไม่ เนื่องจากจุด (0, 0) อยู่บนเส้น เมธอดนี้ก็คืนค่า `True` ด้วย

## ปัญหาทั่วไปและวิธีแก้
| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|--------|--------|
| `line.Covers(point)` คืนค่า `False` แม้ว่าจุดดูเหมือนอยู่บนเส้น | พิกัดจุดไม่ตรงกันอย่างสมบูรณ์เนื่องจากความแม่นยำของจุดลอย | ใช้ `Math.Round` กับพิกัดหรือทำการตรวจสอบด้วย tolerance เช่น `line.Distance(point) < epsilon` |
| ขาด `using Aspose.Gis.Geometries;` | ไม่ได้นำเข้า Namespace ทำให้เกิดข้อผิดพลาดในการคอมไพล์ | ตรวจสอบให้มีคำสั่ง import อยู่ (ดูส่วน **นำเข้า Namespaces**) |
| เกิดข้อยกเว้นเรื่องลิขสิทธิ์ที่ runtime | ไม่ได้โหลดลิขสิทธิ์ที่ถูกต้องสำหรับการใช้งานจริง | โหลดลิขสิทธิ์ชั่วคราวหรือเต็มโดยใช้ `License license = new License(); license.SetLicense("Aspose.GIS.lic");` |
