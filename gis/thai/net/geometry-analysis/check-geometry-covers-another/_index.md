# ตรวจสอบ Geometry ครอบอีกอันหนึ่ง

## บทนำ
ในบทเรียนนี้คุณจะได้เรียนรู้ **วิธีสร้าง linestring c#** ด้วย Aspose.GIS for .NET, เพิ่มจุดลงใน linestring, และทำการตรวจสอบ **จุดบนเส้น** อย่างแม่นยำด้วยเมธอด `Covers` และ `CoveredBy` ไม่ว่าคุณจะกำลังสร้างเครื่องมือแผนที่, ทำการวิเคราะห์เชิงพื้นที่, หรือเพียงต้องการตรวจสอบความสัมพันธ์เชิงเรขาคณิต การเชี่ยวชาญในขั้นตอนเหล่านี้จะทำให้แอปพลิเคชันของคุณมีความแม่นยำที่ต้องการ

## คำตอบสั้น
- **“create linestring c#” หมายถึงอะไร?** หมายถึงการสร้างอ็อบเจ็กต์เรขาคณิต `LineString` และเติมจุดพิกัดลงไป  
- **เมธอดใดใช้ตรวจสอบว่าจุดอยู่บนเส้น?** ใช้เมธอด `Covers` ของ `LineString` หรือ `CoveredBy` ของ `Point`  
- **ต้องมีไลเซนส์เพื่อรันตัวอย่างหรือไม่?** ไลเซนส์ชั่วคราวใช้สำหรับการประเมินผล; ไลเซนส์เต็มจำเป็นสำหรับการใช้งานจริง  
- **สามารถใช้กับ .NET Core ได้หรือไม่?** ใช่, Aspose.GIS รองรับ .NET Framework และ .NET Core  
- **สามารถเพิ่มจุดลงใน linestring ได้กี่จุด?** ไม่มีขีดจำกัดที่แน่นอน; สามารถเพิ่มได้ตามต้องการสำหรับการวิเคราะห์เชิงพื้นที่ของคุณ  

## **create linestring c#** คืออะไร?
`LineString` คือรูปทรงเรขาคณิตที่ประกอบด้วยรายการจุดที่เรียงลำดับกันโดยเชื่อมต่อด้วยส่วนเส้นตรง ใน C# คุณสร้างได้โดยการสร้างอินสแตนซ์ของคลาส `LineString` จากเนมสเปซ `Aspose.Gis.Geometries` แล้ว **เพิ่มจุดลงใน linestring** ด้วยเมธอด `AddPoint`

## ทำไมต้องใช้ Aspose.GIS สำหรับการตรวจสอบจุดบนเส้น?
- **ความแม่นยำ** – จัดการการคำนวณจุดลอยและตัวดำเนินการเชิงพื้นที่อย่างแม่นยำ ให้ผลลัพธ์ **precision point on line** ที่เชื่อถือได้  
- **ข้ามแพลตฟอร์ม** – ทำงานกับ .NET Framework, .NET Core, และ .NET 5/6+  
- **API ครบถ้วน** – มีเมธอดความสัมพันธ์เชิงพื้นที่หลากหลาย (`Covers`, `CoveredBy`, `Intersects` ฯลฯ)

## วิธีสร้าง linestring c# – คู่มือขั้นตอนโดยละเอียด

### ขั้นตอนที่ 1: สร้างอ็อบเจ็กต์ LineString
```csharp
var line = new LineString();
```
ในขั้นตอนนี้เราจะสร้างอินสแตนซ์ใหม่ของ `LineString` ซึ่งแทนลำดับของส่วนเส้นที่เชื่อมต่อกันในพื้นที่สองมิติ

### ขั้นตอนที่ 2: **เพิ่มจุดลงใน LineString**
```csharp
line.AddPoint(0, 0);
line.AddPoint(1, 1);
```
เราจะ **เพิ่มจุดลงใน linestring** ด้วยเมธอด `AddPoint` ในตัวอย่างนี้เราจะเพิ่มสองจุด: (0, 0) และ (1, 1) เพื่อสร้างส่วนเส้นทแยงมุมง่าย ๆ

### ขั้นตอนที่ 3: สร้างอ็อบเจ็กต์ Point
```csharp
var point = new Point(0, 0);
```
สร้างอ็อบเจ็กต์ `Point` ที่แทนจุดเดียวในพื้นที่สองมิติ ที่นี่เราจะสร้างจุดที่พิกัด (0, 0)

### ขั้นตอนที่ 4: ทำ **การตรวจสอบจุดบนเส้น** – เส้นครอบจุดหรือไม่?
```csharp
Console.WriteLine(line.Covers(point));    // True
```
ใช้เมธอด `Covers` เพื่อตรวจสอบว่าเส้นครอบจุดหรือไม่ ในกรณีนี้จะคืนค่า `True` เนื่องจากจุด (0, 0) อยู่บนเส้นอย่างแม่นยำ

### ขั้นตอนที่ 5: ตรวจสอบความสัมพันธ์ย้อนกลับ – จุดถูกเส้นครอบหรือไม่?
```csharp
Console.WriteLine(point.CoveredBy(line)); // True
```
เช่นเดียวกัน, ใช้เมธอด `CoveredBy` เพื่อตรวจสอบว่าจุดถูกเส้นครอบหรือไม่ เนื่องจากจุด (0, 0) อยู่บนเส้น, เมธอดจะคืนค่า `True` ด้วย

## ปัญหาที่พบบ่อยและวิธีแก้
| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|--------|--------|
| `line.Covers(point)` คืนค่า `False` แม้ว่าจุดดูเหมือนอยู่บนเส้น | พิกัดจุดไม่ตรงกันอย่างสมบูรณ์เนื่องจากความแม่นยำของจุดลอย | ใช้ `Math.Round` กับพิกัดหรือทำการตรวจสอบด้วย tolerance เช่น `line.Distance(point) < epsilon` |
| ขาด `using Aspose.Gis.Geometries;` | ไม่ได้นำเข้าเนมสเปซ ทำให้เกิดข้อผิดพลาดในการคอมไพล์ | ตรวจสอบให้มีบรรทัด import (ดูส่วน **นำเข้าเนมสเปซ** ด้านบน) |
