# แปลง TopoJSON เป็น GeoJSON

## บทนำ
ในบทแนะนำนี้ คุณจะได้เรียนรู้ **วิธีการแปลง TopoJSON เป็น GeoJSON** โดยใช้ Aspose.GIS API สำหรับ .NET การแปลงระหว่างรูปแบบข้อมูลภูมิศาสตร์ที่ใช้กันอย่างแพร่หลายสองรูปแบบนี้เป็นความต้องการทั่วไปเมื่อสร้างแผนที่เว็บ, ทำการวิเคราะห์เชิงพื้นที่, หรือรวมข้อมูล GIS เข้ากับแอปพลิเคชัน .NET เราจะเดินผ่านกระบวนการทั้งหมด, อธิบายว่าการแปลงมีความสำคัญอย่างไร, และให้โค้ดตัวอย่างที่พร้อมใช้งาน

## คำตอบอย่างรวดเร็ว
- **การแปลงทำอะไร?** มันแปลงข้อมูลโทโพโลจีของ TopoJSON ให้เป็นคอลเลกชันฟีเจอร์มาตรฐานของ GeoJSON.  
- **ทำไมต้องใช้ Aspose.GIS?** มันให้การเรียก API แบบบรรทัดเดียวที่จัดการงานหนักโดยไม่ต้องใช้เครื่องมือของบุคคลที่สาม.  
- **ใช้เวลานานเท่าไหร่?** การแปลงทั่วไปเสร็จภายในไม่ถึงหนึ่งวินาทีสำหรับไฟล์ขนาดหลายเมกะไบต์.  
- **ต้องการไลเซนส์หรือไม่?** การทดลองใช้ฟรีทำงานสำหรับการพัฒนา; ต้องมีไลเซนส์เชิงพาณิชย์สำหรับการใช้งานจริง.  
- **เวอร์ชัน .NET ที่รองรับคืออะไร?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## “convert topojson to geojson” คืออะไร?
TopoJSON เป็นรูปแบบที่กะทัดรัดซึ่งเก็บส่วนเส้นที่ใช้ร่วมกัน (arc) เพียงครั้งเดียวและอ้างอิงถึงมัน, ทำให้ขนาดไฟล์ลดลง. ในขณะที่ GeoJSON เป็นการแทนค่าข้อมูลภูมิศาสตร์ในรูปแบบ JSON อย่างตรงไปตรงมา. การแปลงทำให้คุณสามารถส่งข้อมูลไปยังไลบรารีที่เข้าใจเฉพาะ GeoJSON — เช่นเฟรมเวิร์กการทำแผนที่ JavaScript จำนวนมาก.

## ทำไมต้องแปลง TopoJSON เป็น GeoJSON?
- **ความเข้ากันได้** – ไลบรารีการทำแผนที่เว็บส่วนใหญ่ (Leaflet, Mapbox GL) คาดหวัง GeoJSON.  
- **ความง่ายในการแก้ไข** – GeoJSON สามารถแก้ไขโดยตรงในโปรแกรมแก้ไขข้อความหรือเครื่องมือ GIS.  
- **การทำงานร่วมกัน** – API และบริการหลายแห่งรับ GeoJSON แต่ไม่รับ TopoJSON.

## คู่มือขั้นตอนต่อขั้นตอน

### ขั้นตอนที่ 1: ระบุเส้นทางอินพุตและเอาต์พุต
กำหนดตำแหน่งที่ไฟล์ TopoJSON ต้นทางอยู่และตำแหน่งที่ไฟล์ GeoJSON ผลลัพธ์จะถูกเขียนออกไป

```csharp
var sampleTopoJsonPath = "Your Document Directory" + "sample.topojson";
var outputFilePath = "Your Document Directory" + "convertedSample_out.geojson";
```

*เคล็ดลับ:* ใช้ `Path.Combine` เพื่อสร้างเส้นทางที่ไม่ขึ้นกับแพลตฟอร์ม

### ขั้นตอนที่ 2: ทำการแปลง
Aspose.GIS ทำงานหนักด้วยการเรียกเมธอดเดียว

```csharp
VectorLayer.Convert(sampleTopoJsonPath, Drivers.TopoJson, outputFilePath, Drivers.GeoJson);
```

หลังจากบรรทัดนี้ทำงานเสร็จ, `convertedSample_out.geojson` จะมีไฟล์ GeoJSON ที่สมบูรณ์ซึ่งคุณสามารถโหลดเข้าโปรแกรมดู GIS ใดก็ได้
