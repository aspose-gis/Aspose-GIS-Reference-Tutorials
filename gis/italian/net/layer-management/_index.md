## Domande frequenti

**Q:** Come posso creare programmaticamente file **create new shapefile**?  
**A:** Usa la classe `Shapefile` in Aspose.GIS, specifica il tipo di geometria, aggiungi i campi, quindi scrivi le feature. Il tutorial collegato “Create New Shapefile” mostra il codice esatto.

**Q:** Qual è la differenza tra un File GDB e uno shapefile?  
**A:** Un File Geodatabase (File GDB) memorizza i dati in una struttura basata su cartelle con supporto a schema più ricco, mentre uno shapefile è un formato semplice e ampiamente compatibile costituito da diversi file a lunghezza fissa.

**Q:** Come posso **filter features by attribute** in modo efficiente?  
**A:** Costruisci un oggetto `Query` con una clausola WHERE (ad esempio, `SELECT * FROM layer WHERE Population > 1000`). La guida “Filter Features by Attribute” ti accompagna attraverso l'API.

**Q:** Posso convertire uno shapefile in GeoJSON senza file intermedi?  
**A:** Sì—carica lo shapefile in una `FeatureCollection` e chiama direttamente `ExportToGeoJson()`. Consulta il tutorial “Extract Features to GeoJSON” per un esempio completo.

**Q:** Ho bisogno di una licenza speciale per eseguire questi esempi localmente?  
**A:** Una licenza di valutazione gratuita è sufficiente per sviluppo e test. Per le distribuzioni in produzione, acquista una licenza commerciale da Aspose.
