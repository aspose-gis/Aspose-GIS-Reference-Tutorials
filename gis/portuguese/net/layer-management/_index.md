## Perguntas Frequentes

**Q:** Como eu **create new shapefile** arquivos programaticamente?  
**A:** Use a classe `Shapefile` no Aspose.GIS, especifique o tipo de geometria, adicione campos e então escreva os recursos. O tutorial vinculado “Create New Shapefile” mostra o código exato.

**Q:** Qual é a diferença entre um File GDB e um shapefile?  
**A:** Um File Geodatabase (File GDB) armazena dados em uma estrutura baseada em pastas com suporte a esquema mais rico, enquanto um shapefile é um formato simples, amplamente compatível, composto por vários arquivos de comprimento fixo.

**Q:** Como posso **filter features by attribute** de forma eficiente?  
**A:** Crie um objeto `Query` com uma cláusula WHERE (por exemplo, `SELECT * FROM layer WHERE Population > 1000`). O guia “Filter Features by Attribute” orienta você através da API.

**Q:** Posso converter um shapefile para GeoJSON sem arquivos intermediários?  
**A:** Sim—carregue o shapefile em uma `FeatureCollection` e chame `ExportToGeoJson()` diretamente. Veja o tutorial “Extract Features to GeoJSON” para um exemplo completo.

**Q:** Preciso de uma licença especial para executar esses exemplos localmente?  
**A:** Uma licença de avaliação gratuita funciona para desenvolvimento e testes. Para implantações em produção, adquira uma licença comercial da Aspose.
