## Questions fréquentes

**Q :** Comment puis‑je créer des fichiers **create new shapefile** programmétiquement ?  
**R :** Utilisez la classe `Shapefile` d’Aspose.GIS, spécifiez le type de géométrie, ajoutez des champs, puis écrivez les entités. Le tutoriel lié « Create New Shapefile » montre le code exact.

**Q :** Quelle est la différence entre un File GDB et un shapefile ?  
**R :** Un File Geodatabase (File GDB) stocke les données dans une structure basée sur des dossiers avec un schéma plus riche, tandis qu’un shapefile est un format simple et largement compatible composé de plusieurs fichiers à longueur fixe.

**Q :** Comment puis‑je **filter features by attribute** efficacement ?  
**R :** Créez un objet `Query` avec une clause WHERE (par ex., `SELECT * FROM layer WHERE Population > 1000`). Le guide « Filter Features by Attribute » vous accompagne dans l’utilisation de l’API.

**Q :** Puis‑je convertir un shapefile en GeoJSON sans fichiers intermédiaires ?  
**R :** Oui — chargez le shapefile dans une `FeatureCollection` et appelez directement `ExportToGeoJson()`. Consultez le tutoriel « Extract Features to GeoJSON » pour un exemple complet.

**Q :** Ai‑je besoin d’une licence spéciale pour exécuter ces exemples localement ?  
**R :** Une licence d’évaluation gratuite suffit pour le développement et les tests. Pour les déploiements en production, achetez une licence commerciale auprès d’Aspose.
