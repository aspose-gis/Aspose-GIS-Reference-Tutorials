# Aspose.GIS का उपयोग करके होल जियोमेट्री के साथ पॉलीगॉन बनाएं

## परिचय
इस ट्यूटोरियल में, आप Aspose.GIS for .NET का उपयोग करके **create polygon with hole** जियोमेट्री बनाएँगे। चाहे आप मैपिंग एप्लिकेशन बना रहे हों, स्पैशियल एनालिसिस कर रहे हों, या GIS सेवाओं के लिए डेटा तैयार कर रहे हों, पॉलीगॉन के अंदर होल एम्बेड करना जानना आवश्यक है। हम पूरी प्रक्रिया को चरण‑दर‑चरण समझेंगे, पर्यावरण सेटअप से लेकर अंतिम पॉलीगॉन ऑब्जेक्ट जनरेट करने तक।

## त्वरित उत्तर
- **What does “create polygon with hole” mean?** यह एक ऐसा पॉलीगॉन बनाना है जिसमें एक या अधिक इंटीरियर रिंग्स (होल) होते हैं जो क्षेत्र से बाहर रखे जाते हैं।  
- **Which library handles this?** Aspose.GIS for .NET बाहरी और आंतरिक रिंग्स के लिए पूर्ण समर्थन प्रदान करता है।  
- **Do I need a license?** विकास के लिए एक फ्री ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **What .NET versions are supported?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **How long does it take?** आमतौर पर लागू करने और परीक्षण करने में 10 मिनट से कम समय लगता है।

## Aspose.GIS का उपयोग करके पॉलीगॉन में होल जोड़ने का तरीका
होल जोड़ना केवल एक **interior ring** को परिभाषित करके और उसे पॉलीगॉन से जोड़कर किया जाता है। लाइब्रेरी अभिविन्यास और वैधता का ध्यान रखती है, इसलिए आप उन निर्देशांक पर ध्यान केंद्रित कर सकते हैं जो आवश्यक खाली स्थान को दर्शाते हैं।

## “create polygon with hole” क्या है?
होल के साथ पॉलीगॉन बनाना एक **exterior ring** को परिभाषित करने में शामिल है जो बाहरी सीमा को रेखांकित करता है और एक या अधिक **interior rings** जो खाली स्थान बनाते हैं। इंटीरियर रिंग्स को अक्सर *होल* कहा जाता है क्योंकि वे पॉलीगॉन की सतह का हिस्सा नहीं होते।

## Aspose.GIS का उपयोग करके पॉलीगॉन में होल क्यों बनाएं?
- **Accurate spatial modeling:** भूमि टुकड़ों के भीतर झीलें या इमारतों के भीतर आंगन जैसे वास्तविक दुनिया के फीचर को होल की आवश्यकता होती है।  
- **Interoperability:** Shapefile, GeoJSON, और GML जैसे फॉर्मेट मूल रूप से इंटीरियर रिंग्स को सपोर्ट करते हैं; Aspose.GIS इन्हें संरक्षित रखता है।  
- **Performance:** लाइब्रेरी जियोमेट्री वैधता को प्रबंधित करती है, इसलिए आपको कस्टम वैधता कोड लिखने की जरूरत नहीं है।

## होल वाले पॉलीगॉन के वास्तविक‑दुनिया परिदृश्य
1. **Land parcel with an internal lake** – झील को एक होल के रूप में मॉडल किया जाता है ताकि वह टुकड़े के क्षेत्र में गिना न जाए।  
2. **Building footprints with courtyards** – आंगन को इमारत के फुटप्रिंट से बाहर रखा जाता है।  
3. **Protected zones inside a larger conservation area** – आप प्रतिबंधित सेक्शन को अलग लेयर बनाए बिना बाहर रख सकते हैं।

## नेमस्पेस आयात करें
सबसे पहले, आपको Aspose.GIS कार्यक्षमताओं के साथ काम करने के लिए आवश्यक नेमस्पेस आयात करने की आवश्यकता है। इसे करने का तरीका इस प्रकार है:

```csharp
using Aspose.Gis.Geometries;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

अब, चलिए Aspose.GIS for .NET का उपयोग करके होल जियोमेट्री के साथ एक पॉलीगॉन बनाते हैं।

## चरण 1: पॉलीगॉन ऑब्जेक्ट बनाएं
हम एक खाली `Polygon` ऑब्जेक्ट बनाकर शुरू करते हैं जो बाद में बाहरी और आंतरिक दोनों रिंग्स को रखेगा।

```csharp
Polygon polygon = new Polygon();
```

## चरण 2: एक्सटीरियर रिंग परिभाषित करें
एक्सटीरियर रिंग पॉलीगॉन की बाहरी सीमा को परिभाषित करती है। बंद आकार बनाने के लिए बिंदुओं को घड़ी की दिशा में जोड़ें।

```csharp
LinearRing ring = new LinearRing();
ring.AddPoint(50.02, 36.22);
ring.AddPoint(49.99, 36.26);
ring.AddPoint(49.97, 36.23);
ring.AddPoint(49.98, 36.17);
ring.AddPoint(50.02, 36.22);
```

## चरण 3: इंटीरियर रिंग (होल) परिभाषित करें
अब, हम एक इंटीरियर रिंग बनाते हैं—यह **hole** पॉलीगॉन के क्षेत्र से बाहर रखा जाएगा। बिंदु सामान्यतः उल्टी‑घड़ी की दिशा में जोड़े जाते हैं, लेकिन Aspose.GIS अभिविन्यास को स्वचालित रूप से संभालता है।

```csharp
LinearRing hole = new LinearRing();
hole.AddPoint(50.00, 36.22);
hole.AddPoint(49.99, 36.20);
hole.AddPoint(49.98, 36.23);
hole.AddPoint(50.00, 36.24);
hole.AddPoint(50.00, 36.22);
```

## चरण 4: एक्सटीरियर रिंग असाइन करें और इंटीरियर रिंग को पॉलीगॉन में जोड़ें
अंत में, एक्सटीरियर रिंग को पॉलीगॉन से जोड़ें और फिर इंटीरियर रिंग (होल) जोड़ें। यदि आपको कई होल चाहिए तो `AddInteriorRing` मेथड को कई बार कॉल किया जा सकता है।

```csharp
polygon.ExteriorRing = ring;
polygon.AddInteriorRing(hole);
```

## टिप्स और सर्वोत्तम अभ्यास
- **Orientation matters for readability** – जबकि Aspose.GIS अभिविन्यास को स्वतः‑सुधारता है, एक्सटीरियर रिंग्स को घड़ी की दिशा में और इंटीरियर रिंग्स को उल्टी‑घड़ी की दिशा में रखना जियोमेट्री को GIS व्यूअर्स में जांचने में आसान बनाता है।  
- **Close each ring** – हमेशा पहले निर्देशांक को अंतिम बिंदु के रूप में दोहराएँ; इससे वैध बंद आकार सुनिश्चित होता है।  
- **Validate after creation** – आप `polygon.IsValid` को कॉल करके जियोमेट्री को सहेजने से पहले OGC मानकों के अनुरूप सुनिश्चित कर सकते हैं।

| समस्या | कारण | समाधान |
