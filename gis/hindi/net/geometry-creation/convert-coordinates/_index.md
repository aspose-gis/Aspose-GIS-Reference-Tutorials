# Aspose.GIS के साथ निर्देशांक को DMS में बदलें

## परिचय
इस ट्यूटोरियल में आप सीखेंगे कि **निर्देशांक को DMS** (डिग्री, मिनट, सेकंड) में कैसे बदलें, Aspose.GIS लाइब्रेरी का उपयोग करके .NET के लिए। चाहे आपको c# में latitude longitude को मैपिंग एप्लिकेशन के लिए बदलना हो, मानव‑पठनीय लोकेशन स्ट्रिंग्स जेनरेट करनी हों, या विभिन्न निर्देशांक फॉर्मैट्स का अन्वेषण करना हो, यह गाइड स्पष्ट व्याख्याओं और तैयार‑चलाने योग्य C# कोड के साथ हर कदम दिखाता है।

## त्वरित उत्तर
- **“निर्देशांक को DMS में बदलना” का क्या मतलब है?** यह संख्यात्मक latitude/longitude मानों को पारंपरिक डिग्री‑मिनट‑सेकंड नोटेशन में बदलता है।  
- **कौन सी लाइब्रेरी परिवर्तन संभालती है?** Aspose.GIS for .NET `GeoConvert` क्लास के साथ अंतर्निहित फॉर्मेट सपोर्ट प्रदान करती है।  
- **क्या इसे आज़माने के लिए लाइसेंस चाहिए?** एक मुफ्त ट्रायल उपलब्ध है; उत्पादन उपयोग के लिए व्यावसायिक लाइसेंस आवश्यक है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, और .NET 5/6+।  
- **क्या मैं वही कोड अन्य फॉर्मैट्स के लिए उपयोग कर सकता हूँ?** हाँ—सिर्फ `PointFormats` enum मान बदलें (जैसे `DecimalDegrees`, `GeoRef`)।  

## निर्देशांक को DMS में बदलना क्या है?
निर्देशांक को DMS में बदलना दशमलव latitude और longitude मानों को `25°30'00"N 45°30'00"E` जैसे फॉर्मेट में पुनर्लेखित करता है। यह प्रतिनिधित्व मानचित्रण, नेविगेशन, और GIS डेटा एक्सचेंज में व्यापक रूप से उपयोग होता है।

## Aspose.GIS को निर्देशांक परिवर्तन के लिए क्यों चुनें?
- **All‑in‑one API** – कोई बाहरी लाइब्रेरी या जटिल गणित नहीं; Aspose.GIS आंतरिक रूप से पार्सिंग और फॉर्मेटिंग संभालता है।  
- **उच्च सटीकता** – नकारात्मक मानों और हेमिस्फ़ेरिक डिज़ाइनेटर्स जैसे किनारे के मामलों को सटीक रूप से संभालता है।  
- **क्रॉस‑प्लेटफ़ॉर्म** – Windows, Linux, और macOS .NET रनटाइम्स पर समान रूप से काम करता है।  
- **विस्तार योग्य** – DMS, दशमलव डिग्री, डिग्री‑डेसिमल‑मिनट, और GeoRef फ स्विच किया जा सकता है।

## नेमस्पेस आयात करें
GIS ऑपरेशन्स के लिए आवश्यक नेमस्पेस आयात करें:

```csharp
using System;

```

## चरण‑दर‑चरण गाइड

### चरण 1: परिवर्तन प्रक्रिया शुरू करें
हम एक मैत्रीपूर्ण संदेश प्रिंट करते हैं ताकि आप जान सकें कि डेमो शुरू हो गया है।

```csharp
Console.WriteLine($"\n== Start: {nameof(ConvertCoordinate)}");
```

### चरण 2: दशमलव डिग्री में बदलें  
भले ही अंतिम लक्ष्य DMS हो, हम मूल दशमलव प्रतिनिधित्व दिखाकर शुरू करते हैं।

```csharp
var decimalDegrees = GeoConvert.AsPointText(25.5, 45.5, PointFormats.DecimalDegrees);
Console.WriteLine(decimalDegrees);
```

### चरण 3: डिग्री डेसिमल मिनट में बदलें  
यह फॉर्मेट (`DD°MM.m'`) एक सामान्य मध्यवर्ती चरण है जब आपको *convert lat long degree minutes* की आवश्यकता होती है।

```csharp
var degreeDecimalMinutes = GeoConvert.AsPointText(25.5, 45.5, PointFormats.DegreeDecimalMinutes);
Console.WriteLine(degreeDecimalMinutes);
```

### चरण 4: डिग्री मिनट सेकंड (DMS) में बदलें  
यह हमारे ट्यूटोरियल का मुख्य भाग है—**निर्देशांक को DMS में बदलें**।

```csharp
var degreeMinutesSeconds = GeoConvert.AsPointText(25.5, 45.5, PointFormats.DegreeMinutesSeconds);
Console.WriteLine(degreeMinutesSeconds);
```

### चरण 5: GeoRef में बदलें  
पूरकता के लिए, हम `GeoRef` फॉर्मेट भी प्रदर्शित करते हैं, जो रिमोट‑सेंसिंग वर्कफ़्लोज़ में उपयोगी है।

```csharp
var geoRef = GeoConvert.AsPointText(25.5, 45.5, PointFormats.GeoRef);
Console.WriteLine(geoRef);
```
