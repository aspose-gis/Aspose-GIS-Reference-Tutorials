# Aspose.GIS के साथ कॉर्डिनेट्स को DMS में कैसे बदलें

## परिचय
इस ट्यूटोरियल में, आप **कोऑर्डिनेट्स को DMS (डिग्री, मिनट, सेकंड)** में बदलना सीखेंगे, जो .NET के लिए शक्तिशाली Aspose.GIS लाइब्रेरी का उपयोग करता है। चाहे आपको **c# convert lat long** करना हो, मानव‑पठनीय लोकेशन स्ट्रिंग्स बनानी हों, या विभिन्न कोऑर्डिनेट फ़ॉर्मेट्स का अन्वेषण करना हो, यह गाइड स्पष्ट व्याख्याओं और तैयार‑चलाने योग्य C# कोड के साथ हर चरण को दर्शाता है।

## त्वरित उत्तर
- **“कोऑर्डिनेट्स को DMS में बदलना” का क्या अर्थ है?** यह संख्यात्मक latitude/longitude मानों को पारंपरिक डिग्री‑मिनट‑सेकंड नोटेशन में परिवर्तित करता है।  
- **कौन सी लाइब्रेरी परिवर्तन संभालती है?** .NET के लिए Aspose.GIS `GeoConvert` क्लास के साथ अंतर्निहित फ़ॉर्मेट सपोर्ट प्रदान करती है।  
- **क्या इसे आज़माने के लिए लाइसेंस चाहिए?** एक मुफ्त ट्रायल उपलब्ध है; उत्पादन उपयोग के लिए व्यावसायिक लाइसेंस आवश्यक है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, और .NET 5/6+।  
- **क्या मैं वही कोड अन्य फ़ॉर्मेट्स के लिए उपयोग कर सकता हूँ?** हाँ—सिर्फ `PointFormats` enum मान बदलें (जैसे `DecimalDegrees`, `GeoRef`)।  

## कोऑर्डिनेट्स को DMS में कैसे बदलें
कोड में डुबकी लगाने से पहले, यह स्पष्ट करते हैं कि DMS अभी भी क्यों महत्वपूर्ण है। मानचित्रकार, पायलट, और कई GIS डेटा प्रदाता DMS को पसंद करते हैं क्योंकि यह मानव‑मित्रवत है और पारंपरिक मानचित्रों के साथ संरेखित है। Aspose.GIS मैन्युअल गणना की आवश्यकता को समाप्त करता है, जिससे आप अपने एप्लिकेशन लॉजिक पर ध्यान केंद्रित कर सकते हैं।

## कोऑर्डिनेट्स को DMS में बदलना क्या है?
कोऑर्डिनेट्स को DMS में बदलना दशमलव latitude और longitude मानों को `25°30'00"N 45°30'00"E` जैसे फ़ॉर्मेट में पुनर्लेखित करता है। यह प्रतिनिधित्व मानचित्रण, नेविगेशन, और GIS डेटा एक्सचेंज में व्यापक रूप से उपयोग होता है।

## कोऑर्डिनेट परिवर्तन के लिए Aspose.GIS क्यों उपयोग करें?
- **ऑल‑इन‑वन API** – बाहरी लाइब्रेरी या जटिल गणित की आवश्यकता नहीं; Aspose.GIS आंतरिक रूप से पार्सिंग और फ़ॉर्मेटिंग संभालता है।  
- **उच्च सटीकता** – नकारात्मक मानों और हेमिस्फ़ेरिक डिज़ाइनेटर्स जैसे किनारे के मामलों को सटीक रूप से संभालता है।  
- **क्रॉस‑प्लेटफ़ॉर्म** – Windows, Linux, और macOS .NET रनटाइम्स पर समान रूप से काम करता है।  
- **विस्तार योग्य** – DMS, दशमलव डिग्री, डिग्री‑डेसिमल‑मिनट, और GeoRef फ़ॉर्मेट्स के बीच आसानी से स्विच कर सकते हैं।

## नेमस्पेस इम्पोर्ट करें
GIS ऑपरेशन्स के लिए आवश्यक नेमस्पेस इम्पोर्ट करें:

```csharp
using System;
using Aspose.Gis;
```

## चरण‑दर‑चरण गाइड

### चरण 1: परिवर्तन प्रक्रिया शुरू करें
हम एक मैत्रीपूर्ण संदेश प्रिंट करते हैं ताकि आपको पता चले कि डेमो शुरू हो गया है।

```csharp
Console.WriteLine($"\n== Start: {nameof(ConvertCoordinate)}");
```

### चरण 2: दशमलव डिग्री में बदलें  
हालांकि अंतिम लक्ष्य DMS है, हम मूल दशमलव प्रतिनिधित्व दिखाकर शुरू करते हैं। यह **decimal degrees to dms** पाथ को भी प्रदर्शित करता है जिसे आप बाद में अनुसरण करेंगे।

```csharp
var decimalDegrees = GeoConvert.AsPointText(25.5, 45.5, PointFormats.DecimalDegrees);
Console.WriteLine(decimalDegrees);
```

### चरण 3: डिग्री डेसिमल मिनट में बदलें  
यह फ़ॉर्मेट (`DD°MM.m'`) एक सामान्य मध्यवर्ती चरण है जब आपको *convert lat long degree minutes* की आवश्यकता होती है।

```csharp
var degreeDecimalMinutes = GeoConvert.AsPointText(25.5, 45.5, PointFormats.DegreeDecimalMinutes);
Console.WriteLine(degreeDecimalMinutes);
```

### चरण 4: डिग्री मिनट सेकंड (DMS) में बदलें  
यह हमारे ट्यूटोरियल का मुख्य भाग है—**convert coordinates to DMS**।

```csharp
var degreeMinutesSeconds = GeoConvert.AsPointText(25.5, 45.5, PointFormats.DegreeMinutesSeconds);
Console.WriteLine(degreeMinutesSeconds);
```

### चरण 5: GeoRef में बदलें  
पूरकता के लिए, हम `GeoRef` फ़ॉर्मेट भी दिखाते हैं, जो रिमोट‑सेंसिंग वर्कफ़्लोज़ में उपयोगी है।

```csharp
var geoRef = GeoConvert.AsPointText(25.5, 45.5, PointFormats.GeoRef);
Console.WriteLine(geoRef);
```
