# GeoTIFF को PNG में बदलें Aspose.GIS for .NET के साथ

## परिचय
यदि आपको **convert GeoTIFF to PNG** तेज़ी से और रंग मानचित्रण पर पूर्ण नियंत्रण के साथ चाहिए, तो आप सही जगह पर हैं। इस ट्यूटोरियल में हम पूरी कार्यप्रवाह को समझेंगे—GeoTIFF फ़ाइलों को लोड करना, कस्टम colourizers लागू करना, और परिणाम को PNG या SVG के रूप में निर्यात करना। चाहे आप वेब‑आधारित मानचित्र सेवा, डेस्कटॉप GIS व्यूअर, या स्वचालित डेटा‑प्रोसेसिंग पाइपलाइन बना रहे हों, ये कदम आपको किसी भी .NET प्लेटफ़ॉर्म पर उच्च‑गुणवत्ता वाले रास्टर विज़ुअलाइज़ेशन के लिए एक ठोस, प्रोडक्शन‑रेडी आधार प्रदान करते हैं।

## त्वरित उत्तर
- **Can Aspose.GIS convert GeoTIFF to PNG?** हाँ – `Map.Render` को `Renderers.Png` के साथ कॉल करें और लाइब्रेरी प्रोजेक्शन और colour mapping को स्वचालित रूप से संभालती है।  
- **What format can I export the map to besides PNG?** उसी मानचित्र का स्केलेबल वेक्टर संस्करण निर्यात करने के लिए `Renderers.Svg` का उपयोग करें।  
- **Do I need a licence for raster rendering?** मूल्यांकन के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन डिप्लॉयमेंट के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **Which .NET versions are supported?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7+.  
- **Is colour‑band manipulation possible?** बिल्कुल – `MultiBandColor` colourizer आपको व्यक्तिगत रास्टर बैंड को RGB चैनलों में मैप करने देता है।

## “convert GeoTIFF to PNG” क्या है?
GeoTIFF को PNG में बदलना एक जियोरेफ़रेंस्ड रास्टर को हल्के PNG चित्र में परिवर्तित करता है।  
यह प्रक्रिया दृश्य प्रतिनिधित्व को संरक्षित रखती है जबकि भारी जियोस्पेशियल मेटाडेटा को हटा देती है, जिससे परिणाम वेब ब्राउज़रों और मोबाइल ऐप्स के लिए आदर्श बन जाता है। Aspose.GIS प्रोजेक्शन हैंडलिंग, colour mapping, और फ़ाइल‑फ़ॉर्मेट ट्रांसलेशन को एक ही कॉल में करता है, इसलिए आपको बाहरी टूल्स की आवश्यकता नहीं है।

## GeoTIFF को PNG में कैसे बदलें?
RasterLayer एक क्लास है जो रास्टर डेटासेट का प्रतिनिधित्व करती है और इसे मानचित्र में जोड़ा जा सकता है। अपने GeoTIFF को `new RasterLayer("input.tif")` से लोड करें, `MultiBandColor` colourizer लागू करें (जो तीन बैंड तक को RGB चैनलों में मैप करता है), और `map.Render("output.png", Renderers.Png)` को कॉल करें। यह एक‑लाइन रेंडरिंग पाइपलाइन स्रोत रास्टर को वेब‑रेडी PNG में बदल देती है जबकि रंग की सटीकता और स्थानिक सीमा को संरक्षित रखती है।

पहला उदाहरण दिखाता है कि कैसे GeoTIFF को लोड करें, एक कस्टम colourizer लागू करें, और **convert GeoTIFF to PNG**। आउटपुट फ़ाइल एक मानक PNG है जिसे किसी भी ब्राउज़र में प्रदर्शित किया जा सकता है।

### चरण 2: पोलर रास्टर ड्रॉ करें (GeoTIFF → PNG)
इस उदाहरण में, हम एक कस्टम colourizer का उपयोग करके बेहतर प्रदर्शन के लिए एक पोलर रास्टर मानचित्र बनाएंगे।
```csharp
var colorizer = new MultiBandColor()
{
    RedBand = new BandColor() { BandIndex = 0, Min = 0, Max = 255 },
    GreenBand = new BandColor() { BandIndex = 1, Min = 0, Max = 255 },
    BlueBand = new BandColor() { BandIndex = 2, Min = 0, Max = 255 }
};
using (var map = new Map(500, 500))
{
    map.SpatialReferenceSystem = SpatialReferenceSystem.CreateFromEpsg(102034);
    map.Extent = new Extent(-180, 60, 180, 90) { SpatialReferenceSystem = SpatialReferenceSystem.Wgs84 };
    map.BackgroundColor = Color.Azure;
    var layer = Drivers.GeoTiff.OpenLayer(Path.Combine(dataDir, "raster_countries.tif"));
    map.Add(layer, colorizer);
    map.Render(dataDir + "raster_countries_gnomonic_out.png", Renderers.Png);
}
```

## मानचित्र को SVG के रूप में कैसे निर्यात करें?
Renderers.Svg एक enum मान है जो इंजन को Scalable Vector Graphics आउटपुट करने का निर्देश देता है। SVG के रूप में निर्यात करना रेंडरर बदलने जितना सरल है: मानचित्र सीमा और colourizer कॉन्फ़िगर करने के बाद `map.Render("output.svg", Renderers.Svg)` को कॉल करें। परिणामी SVG रिज़ॉल्यूशन‑इंडिपेंडेंट है, जिससे यह प्रिंट‑क्वालिटी मानचित्र या इंटरैक्टिव वेब विज़ुअलाइज़ेशन के लिए उपयुक्त बनता है।

अब, चलिए स्वचालित colour detection और रैखिक इंटरपोलेशन के साथ एक विकृत रास्टर मानचित्र बनाते हैं, और परिणाम को SVG के रूप में निर्यात करते हैं।
```csharp
using (var map = new Map(500, 500))
{
    map.BackgroundColor = Color.Azure;
    var layer = Drivers.GeoTiff.OpenLayer(Path.Combine(dataDir, "raster_skew.tif"));
    map.Add(layer);
    map.Render(dataDir + "raster_skew_out.svg", Renderers.Svg);
}
```

## सामान्य समस्याएँ और समाधान
| Issue | Solution |
|-------|----------|
| **आउटपुट इमेज खाली है** | `dataDir` सही फ़ोल्डर की ओर इशारा करता है और GeoTIFF फ़ाइलें मौजूद हैं, यह सत्यापित करें। |
| **रंग फीके दिख रहे हैं** | `MultiBandColor` में `Min`/`Max` मानों को प्रत्येक बैंड की डेटा रेंज से मिलाने के लिए समायोजित करें। |
| **प्रोजेक्शन मेल नहीं खाता** | सुनिश्चित करें कि मानचित्र का `SpatialReferenceSystem` स्रोत रास्टर से मेल खाता है या `SpatialReferenceSystem.CreateFromEpsg` का उपयोग करके पुनः‑प्रोजेक्ट करें। |
| **SVG फ़ाइल बहुत बड़ी है** | रेंडरिंग से पहले ज्योमेट्री को सरल बनाने या मानचित्र सीमा को सीमित करने के लिए `RenderOptions` का उपयोग करें। |
