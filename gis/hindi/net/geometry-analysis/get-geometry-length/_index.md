# Aspose.GIS के साथ .NET में ज्यामिति लंबाई कैसे गणना करें

## परिचय
यदि आप **calculate geometry length .NET** के लिए एक स्पष्ट, व्यावहारिक तरीका खोज रहे हैं, तो आप सही जगह पर आए हैं। Aspose.GIS for .NET आपको GIS‑केंद्रित APIs का एक समृद्ध सेट प्रदान करता है जो स्थानिक गणनाओं—जैसे लाइन लंबाई या बहुभुज परिधि मापना—को सरल और तेज बनाता है। इस ट्यूटोरियल में हम पूरी प्रक्रिया को चरण-दर-चरण देखेंगे, पर्यावरण सेटअप से लेकर C# कोड लिखने तक जो सटीक लंबाई मान लौटाता है।

## त्वरित उत्तर
- **GetLength() क्या लौटाता है?** लाइन के लिए यह लाइन लंबाई लौटाता है; बहुभुज के लिए यह परिधि लौटाता है।  
- **कौन सा नेमस्पेस आवश्यक है?** `Aspose.Gis.Geometries`।  
- **क्या मैं इसे .NET 6 के साथ उपयोग कर सकता हूँ?** हाँ, Aspose.GIS .NET 5, .NET 6, और बाद के संस्करणों को सपोर्ट करता है।  
- **क्या विकास के लिए लाइसेंस चाहिए?** मूल्यांकन के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए लाइसेंस आवश्यक है।  
- **क्या गणना इकाई‑सचेत है?** लंबाई कोऑर्डिनेट सिस्टम की इकाइयों में लौटाई जाती है (जैसे, प्रोजेक्टेड CRS के लिए मीटर)।

## ज्यामिति लंबाई क्या है?
`Geometry.GetLength()` एक मेथड है जो ज्यामिति ऑब्जेक्ट की रैखिक माप लौटाता है। `LineString` के लिए यह कुल लाइन लंबाई देता है, जबकि `Polygon` के लिए यह परिधि (सभी किनारों का योग) लौटाता है। यह मेथड अंतर्निहित गणित को सारगर्भित करता है, जिससे आप उच्च‑स्तरीय GIS लॉजिक पर ध्यान केंद्रित कर सकते हैं।

## लंबाई गणनाओं के लिए Aspose.GIS क्यों उपयोग करें?
- **कोई बाहरी निर्भरताएँ नहीं** – शुद्ध .NET लाइब्रेरी, कोई नेटिव DLL नहीं।  
- **उच्च सटीकता** – सटीक परिणामों के लिए डबल‑प्रिसिशन अंकगणित का उपयोग करता है।  
- **क्रॉस‑प्लेटफ़ॉर्म** – Windows, Linux, और macOS पर .NET Core/5/6+ के साथ काम करता है।

## नेमस्पेस आयात करें
Aspose.GIS for .NET द्वारा प्रदान की गई कार्यक्षमताओं का उपयोग करने के लिए, आपको अपने C# प्रोजेक्ट में आवश्यक नेमस्पेस आयात करने होंगे।

### Aspose.GIS नेमस्पेस आयात करें
```csharp
using Aspose.Gis.Geometries;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

## C# में लाइन लंबाई कैसे प्राप्त करें
### चरण 1: ज्यामिति ऑब्जेक्ट बनाएं
शुरुआत करने के लिए, उन आकारों के ज्यामिति ऑब्जेक्ट बनाएं जिनकी लंबाई आप गणना करना चाहते हैं। इसमें लाइन, बहुभुज, या कोई भी अन्य ज्यामितीय आकार शामिल हो सकते हैं।

```csharp
var line = new LineString();
line.AddPoint(0, 0);
line.AddPoint(2, 2);
line.AddPoint(2, 0);
```

### चरण 2: C# में लाइन लंबाई गणना करें
एक बार जब आप लाइन ज्यामिति बना लेते हैं, तो आप `GetLength()` मेथड का उपयोग करके उसकी लंबाई गणना कर सकते हैं। यह एक ही कोड लाइन में **calculate line length c#** को दर्शाता है।

```csharp
Console.WriteLine("{0:F}", line.GetLength()); // Output: 4.83
```

## बहुभुजों के लिए C# में लाइन लंबाई कैसे गणना करें
### चरण 3: बहुभुज ज्यामिति बनाएं
इसी तरह, आप `Polygon` और `LinearRing` क्लासों का उपयोग करके बहुभुज ज्यामिति ऑब्जेक्ट बना सकते हैं।

```csharp
var rectangle = new Polygon(new LinearRing(new[]
{
    new Point(0, 0),
    new Point(0, 1),
    new Point(1, 1),
    new Point(1, 0),
    new Point(0, 0),
}));
```

### चरण 4: बहुभुज की लंबाई प्राप्त करें
बहुभुजों के लिए, `GetLength()` मेथड परिधि लौटाता है, जो प्रभावी रूप से आकार की **how to get length** है।

```csharp
Console.WriteLine("{0:F}", rectangle.GetLength()); // Output: 4.00
```
