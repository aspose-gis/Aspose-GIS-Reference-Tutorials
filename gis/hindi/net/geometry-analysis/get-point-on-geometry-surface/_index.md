# बहुभुज के भीतर बिंदु जाँचें और सतह पर बिंदु प्राप्त करें

## परिचय
इस ट्यूटोरियल में आप Aspose.GIS for .NET के साथ **बहुभुज के भीतर बिंदु जाँचने** का तरीका सीखेंगे और यह भी देखेंगे कि **ज्यामिति की सतह पर बिंदु** कैसे प्राप्त किया जाता है। हम C# में एक बहुभुज बनाने, बहुभुज की सतह पर स्थित बिंदु प्राप्त करने, और यह सत्यापित करने की प्रक्रिया को समझेंगे कि वह बिंदु वास्तव में बहुभुज के भीतर है। अंत तक, आपके पास एक तैयार‑से‑उपयोग स्निपेट होगा जिसे आप किसी भी .NET जियोस्पेशियल एप्लिकेशन में जोड़ सकते हैं।

## त्वरित उत्तर
- **“बहुभुज के भीतर बिंदु जाँचें” का क्या अर्थ है?** यह सत्यापित करता है कि दिया गया निर्देशांक बहुभुज ज्यामिति की सीमाओं के भीतर स्थित है या नहीं।  
- **कौन सा मेथड बहुभुज के अंदर का बिंदु लौटाता है?** `GetPointOnSurface()` एक ऐसा बिंदु लौटाता है जो निश्चित रूप से बहुभुज के भीतर होता है।  
- **क्या उदाहरण चलाने के लिए लाइसेंस चाहिए?** मूल्यांकन के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए पूर्ण लाइसेंस आवश्यक है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework, .NET Core, और .NET Standard सभी संगत हैं।  
- **इम्प्लीमेंटेशन में कितना समय लगेगा?** कॉपी, कंपाइल और रन करने में लगभग 5‑10 मिनट लगते हैं।

2. अपने विकास पर्यावरण को सेट अप करें: सुनिश्चित करें कि आपके पास .NET प्रोग्रामिंग के लिए एक कार्यशील विकास पर्यावरण है। यदि नहीं, तो आप Visual Studio या अपनी पसंद का कोई अन्य .NET विकास पर्यावरण सेट कर सकते हैं।  
3. C# का मूल ज्ञान: यदि आप पहले से परिचित नहीं हैं तो C# प्रोग्रामिंग भाषा की बुनियादी बातों से परिचित हों।  
4. दस्तावेज़ीकरण तक पहुंच: ट्यूटोरियल के दौरान संदर्भ के लिए [दस्तावेज़ीकरण](https://reference.aspose.com/gis/net/) को हाथ में रखें।

## नेमस्पेस इम्पोर्ट करें
इम्प्लीमेंटेशन में गहराई से जाने से पहले, आवश्यक नेमस्पेस को इम्पोर्ट करके शुरू करते हैं:

```csharp
using Aspose.Gis.Geometries;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

अब जबकि हमने अपना पर्यावरण सेट कर लिया है और आवश्यक नेमस्पेस इम्पोर्ट कर लिए हैं, चलिए उदाहरण को कई चरणों में विभाजित करके इसे बेहतर समझते हैं।

## C# में बहुभुज कैसे बनाएं
सबसे पहले, हमें **बहुभुज** ज्यामिति बनानी होगी। हम बहुभुज की बाहरी रिंग को उसके शीर्ष बिंदुओं को निर्दिष्ट करके परिभाषित करते हैं।

```csharp
var polygon = new Polygon();
polygon.ExteriorRing = new LinearRing(new[]
{
    new Point(0, 0),
    new Point(0, 1),
    new Point(1, 1),
    new Point(0, 0),
});
```

## सतह पर बिंदु कैसे प्राप्त करें
अगला, हम `GetPointOnSurface()` मेथड का उपयोग करके बहुभुज की सतह पर एक बिंदु प्राप्त करते हैं। यह **सतह पर बिंदु प्राप्त करने** चरण है।

```csharp
IPoint pointOnSurface = polygon.GetPointOnSurface();
```

## बहुभुज के भीतर बिंदु कैसे जाँचें
हम `SpatiallyContains()` मेथड का उपयोग करके यह सत्यापित कर सकते हैं कि प्राप्त बिंदु बहुभुज के भीतर स्थित है या नहीं। यह **बहुभुज पर बिंदु प्राप्त करना** और फिर उसे जाँचने का प्रदर्शन करता है।

```csharp
Console.WriteLine(polygon.SpatiallyContains(pointOnSurface)); // True
```

## सामान्य समस्याएँ और समाधान
- **खाली बहुभुज** – सुनिश्चित करें कि बाहरी रिंग में कम से कम तीन अलग-अलग शीर्ष बिंदु हों; अन्यथा `GetPointOnSurface()` एक अनिर्धारित बिंदु लौटा सकता है।  
- **घड़ी की दिशा बनाम विपरीत दिशा** – रिंग की अभिविन्यास containment check को प्रभावित नहीं करता, लेकिन एकसमान winding order अन्य स्पेशियल ऑपरेशनों में मदद करता है।  
- **निर्देशांक प्रणाली** – उदाहरण एक सरल कार्टेशियन प्लेन का उपयोग करता है; वास्तविक विश्व निर्देशांकों के साथ काम करते समय सुनिश्चित करें कि CRS (coordinate reference system) सही ढंग से परिभाषित हो।
