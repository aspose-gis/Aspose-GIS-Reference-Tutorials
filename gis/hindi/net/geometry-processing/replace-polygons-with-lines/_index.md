# Aspose.GIS for .NET के साथ बहुभुज को रेखा में बदलें

## परिचय
यदि आपको .NET GIS प्रोजेक्ट में **convert polygon to line** की आवश्यकता है, तो Aspose.GIS प्रक्रिया को सरल बनाता है। चाहे आप मानचित्र दृश्य को सरल बना रहे हों, रूटिंग एल्गोरिदम के लिए डेटा तैयार कर रहे हों, या केवल एक साफ़ ज्यामिति प्रतिनिधित्व चाहिए, यह ट्यूटोरियल आपको Aspose.GIS API का उपयोग करके बहुभुज को रेखा ज्यामितियों में बदलने के चरणों के माध्यम से ले जाएगा।

## त्वरित उत्तर
- **“convert polygon to line” क्या है?** यह बंद बहुभुज आकारों को उनकी सीमा रेखा स्ट्रिंग में बदल देता है।  
- **इस कार्य के लिए Aspose.GIS क्यों उपयोग करें?** यह एक ही मेथड (`ReplacePolygonsByLines`) प्रदान करता है जो मैन्युअल ज्यामिति पार्सिंग के बिना प्रभावी रूप से रूपांतरण को संभालता है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, और .NET 5/6+.  
- **क्या विकास के लिए लाइसेंस आवश्यक है?** परीक्षण के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **कार्यान्वयन में कितना समय लगता है?** सामान्यतः बुनियादी रूपांतरण के लिए 10 मिनट से कम।

## “convert polygon to line” क्या है?
बहुभुज को रेखा में बदलना का अर्थ है बहुभुज की बाहरी रिंग (परिधि) को निकालना और उसे `LineString` के रूप में प्रस्तुत करना। परिणामी ज्यामिति आकार की रूपरेखा को बरकरार रखती है लेकिन आंतरिक क्षेत्र की जानकारी खो देती है, जो नेटवर्क विश्लेषण या किनारा रेंडरिंग जैसे कार्यों के लिए उपयोगी है।

## Aspose.GIS के साथ बहुभुज को रेखा में बदलने का कारण?
- **दृश्य को सरल बनाएं:** रेखाएँ रेंडर करने में हल्की होती हैं, विशेषकर वेब मानचित्रों पर।  
- **रूटिंग के लिए डेटा तैयार करें:** कई रूटिंग इंजन को रेखा ज्यामिति की आवश्यकता होती है।  
- **टोपोलॉजी बनाए रखें:** रेखा मूल बहुभुज की सटीक सीमा को बरकरार रखती है, जिससे स्थानिक सटीकता सुनिश्चित होती है।  
- **एक‑लाइन समाधान:** `ReplacePolygonsByLines()` मेथड आपके लिए सभी जटिल कार्य करता है।

## नेमस्पेस आयात करें
अपने .NET प्रोजेक्ट में, आवश्यक नेमस्पेस आयात करें ताकि आप Aspose.GIS क्लासेस के साथ काम कर सकें।

```csharp
using System;
using Aspose.Gis.Geometries;
```

## चरण‑दर‑चरण मार्गदर्शिका

### चरण 1: स्रोत ज्यामिति निर्धारित करें
एक ज्यामिति संग्रह बनाएं जिसमें एक या अधिक बहुभुज शामिल हों जिन्हें आप बदलना चाहते हैं। इस उदाहरण में हम एक बिंदु भी जोड़ते हैं ताकि दिखाया जा सके कि गैर‑बहुभुज तत्व अपरिवर्तित रहते हैं।

```csharp
var srcGeometry = Geometry.FromText(@"GeometryCollection (POLYGON((1 2, 1 4, 3 4, 3 2)), Point (5 1))");
```

### चरण 2: बहुभुज को रेखा में बदलें
`ReplacePolygonsByLines()` मेथड को कॉल करें। यह एकल कॉल संग्रह को स्कैन करता है, प्रत्येक बहुभुज को उसकी संबंधित रेखा प्रतिनिधित्व से बदल देता है, और अन्य ज्यामिति प्रकारों को अपरिवर्तित छोड़ देता है।

```csharp
var dstGeometry = srcGeometry.ReplacePolygonsByLines();
```

### चरण 3: मूल और परिवर्तित ज्यामितियों को प्रदर्शित करें
कंसोल में मूल और परिवर्तित दोनों ज्यामितियों को प्रिंट करें ताकि आप रूपांतरण की पुष्टि कर सकें।

```csharp
Console.WriteLine($"source: {srcGeometry.AsText()}");
Console.WriteLine($"result: {dstGeometry.AsText()}");
```
