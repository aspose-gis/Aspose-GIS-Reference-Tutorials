## Часто задаваемые вопросы

**Q:** Как я могу программно **создать новые shapefile** файлы?  
**A:** Используйте класс `Shapefile` в Aspose.GIS, укажите тип геометрии, добавьте поля, затем запишите объекты. Связанное руководство «Create New Shapefile» показывает точный код.

**Q:** В чём разница между File GDB и shapefile?  
**A:** Файловая геодатабаза (File GDB) хранит данные в структуре на основе папок с более богатой поддержкой схем, тогда как shapefile — простой, широко совместимый формат, состоящий из нескольких файлов фиксированной длины.

**Q:** Как я могу эффективно **фильтровать объекты по атрибуту**?  
**A:** Создайте объект `Query` с условием WHERE (например, `SELECT * FROM layer WHERE Population > 1000`). Руководство «Filter Features by Attribute» проведёт вас через API.

**Q:** Можно ли преобразовать shapefile в GeoJSON без промежуточных файлов?  
**A:** Да — загрузите shapefile в `FeatureCollection` и вызовите `ExportToGeoJson()` напрямую. Смотрите руководство «Extract Features to GeoJSON» для полного примера.

**Q:** Нужна ли специальная лицензия для локального запуска этих примеров?  
**A:** Бесплатная оценочная лицензия подходит для разработки и тестирования. Для продакшн‑развёртываний приобретите коммерческую лицензию у Aspose.
