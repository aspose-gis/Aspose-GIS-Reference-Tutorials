# Преобразование TopoJSON в GeoJSON

## Введение
В этом руководстве вы узнаете **как преобразовать TopoJSON в GeoJSON** с помощью Aspose.GIS API для .NET. Преобразование между этими двумя широко используемыми форматами геоданных часто требуется при создании веб‑карт, выполнении пространственного анализа или интеграции GIS‑данных в .NET‑приложения. Мы пройдём весь процесс, объясним, почему это преобразование важно, и предоставим готовые к запуску фрагменты кода.

## Быстрые ответы
- **Что делает преобразование?** Оно преобразует топологические данные TopoJSON в стандартные коллекции объектов GeoJSON.  
- **Почему использовать Aspose.GIS?** Он предоставляет однострочный вызов API, который выполняет всю тяжёлую работу без сторонних инструментов.  
- **Сколько это занимает времени?** Обычные преобразования завершаются менее чем за секунду для файлов размером до нескольких мегабайт.  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для разработки; коммерческая лицензия требуется для продакшн‑использования.  
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

1. **Aspose.GIS for .NET** – скачайте и установите последнюю библиотеку с [Aspose.GIS website](https://releases.aspose.com/gis/net/).

## Что такое «convert topojson to geojson»?
TopoJSON — это компактный формат, который хранит общие линейные сегменты (дуги) один раз и ссылается на них, что уменьшает размер файла. GeoJSON, с другой стороны, представляет географические объекты в виде простого JSON. Преобразование позволяет передать данные в библиотеки, которые понимают только GeoJSON — такие как многие JavaScript‑фреймворки для карт.

## Почему преобразовывать TopoJSON в GeoJSON?
- **Совместимость** — большинство веб‑картографических библиотек (Leaflet, Mapbox GL) ожидают GeoJSON.  
- **Удобство редактирования** — GeoJSON можно редактировать напрямую в текстовых редакторах или GIS‑инструментах.  
- **Интероперабельность** — многие API и сервисы принимают GeoJSON, но не TopoJSON.

## Пошаговое руководство

### Шаг 1: Укажите пути ввода и вывода
Определите, где находится исходный TopoJSON и куда следует записать полученный GeoJSON.

```csharp
var sampleTopoJsonPath = "Your Document Directory" + "sample.topojson";
var outputFilePath = "Your Document Directory" + "convertedSample_out.geojson";
```

*Совет:* используйте `Path.Combine` для построения пути, независимого от платформы.

### Шаг 2: Выполните преобразование
Aspose.GIS выполняет всю тяжёлую работу одним вызовом метода.

```csharp
VectorLayer.Convert(sampleTopoJsonPath, Drivers.TopoJson, outputFilePath, Drivers.GeoJson);
```

После выполнения этой строки файл `convertedSample_out.geojson` будет содержать полностью корректный GeoJSON, который можно загрузить в любой GIS‑просмотрщик.
