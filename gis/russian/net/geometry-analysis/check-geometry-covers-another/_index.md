# Проверка того, что одна геометрия покрывает другую

## Введение
В этом руководстве вы узнаете, **как создать LineString в C#** с помощью Aspose.GIS для .NET, добавить точки в LineString и выполнить надёжную **проверку принадлежности точки линии** с помощью методов `Covers` и `CoveredBy`. Независимо от того, создаёте ли вы инструмент картографирования, проводите пространственный анализ или просто хотите проверить геометрические отношения, освоение этих операций придаст вашему приложению необходимую точность.

- **Какой метод проверяет, лежит ли точка на линии?** Используйте метод `Covers` у `LineString` или `CoveredBy` у `Point`.

## Как создать linestring c# – Пошаговое руководство

### Шаг 1: Создайте объект LineString
```csharp
var line = new LineString();
```
Здесь мы инстанцируем новый объект `LineString`, представляющий последовательность соединённых отрезков в двумерном пространстве.

### Шаг 2: **Добавьте точки в LineString**
```csharp
line.AddPoint(0, 0);
line.AddPoint(1, 1);
```
Мы **добавляем точки в LineString** методом `AddPoint`. В примере добавляются две точки: (0, 0) и (1, 1), образующие простой диагональный отрезок.

### Шаг 3: Создайте объект Point
```csharp
var point = new Point(0, 0);
```
Инстанцируем объект `Point`, представляющий одну точку в двумерном пространстве. Здесь создаётся точка с координатами (0, 0).

### Шаг 4: Выполните **проверку принадлежности точки линии** – покрывает ли линия точку?
```csharp
Console.WriteLine(line.Covers(point));    // True
```
Используем метод `Covers`, чтобы проверить, покрывает ли линия точку. В данном случае он возвращает `True`, поскольку точка (0, 0) лежит точно на линии.

### Шаг 5: Проверьте обратное отношение – покрывается ли точка линией?
```csharp
Console.WriteLine(point.CoveredBy(line)); // True
```
Аналогично, используем метод `CoveredBy`, чтобы проверить, покрывается ли точка линией. Поскольку точка (0, 0) находится на линии, метод также возвращает `True`.

## Распространённые проблемы и решения
| Проблема | Почему возникает | Решение |
|----------|------------------|---------|
| `line.Covers(point)` возвращает `False`, хотя точка выглядит как лежащая на линии | Координаты точки не совпадают точно из‑за погрешности плавающей точки | Округлите координаты с помощью `Math.Round` или выполните проверку с допуском: `line.Distance(point) < epsilon`. |
| Отсутствует `using Aspose.Gis.Geometries;` | Пространство имён не импортировано, что приводит к ошибкам компиляции | Убедитесь, что оператор импорта присутствует (см. раздел **Импорт пространств имён**). |
| Исключение лицензии во время выполнения | Не загружена действующая лицензия для продакшна | Загрузите временную или полную лицензию: `License license = new License(); license.SetLicense("Aspose.GIS.lic");`. |
