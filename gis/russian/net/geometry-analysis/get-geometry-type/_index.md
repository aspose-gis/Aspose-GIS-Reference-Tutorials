# Получить тип геометрии с помощью Aspose.GIS для .NET

## Введение
Если вам нужно **получить тип геометрии** пространственного объекта в приложении .NET, Aspose.GIS делает это проще простого. В этом руководстве мы пройдём весь процесс — от настройки вашей GIS‑среды до создания объекта точки и извлечения его типа геометрии. К концу вы поймёте, как **работать с пространственными данными** эффективно и сможете расширить пример до линий, полигонов и прочего.

## Быстрые ответы
- **Что означает «получить тип геометрии»?** Возвращает значение перечисления (например, Point, LineString), которое идентифицирует форму объекта геометрии.  
- **Какая библиотека предоставляет эту возможность?** Aspose.GIS для .NET.

- **Сколько времени занимает настройка?** Обычно менее 10 минут для базового примера с точкой.

## Что такое «получить тип геометрии» в Aspose.GIS?
`GeometryType` — это перечисление, описывающее тип геометрии, с которой вы работаете: Point, LineString, Polygon и т.д. Доступ к этому свойству позволяет принимать решения в коде на основе формы обрабатываемых данных.

## Как получить тип геометрии из точки
Ниже представлен **пример с точкой .net**, демонстрирующий полный процесс — от создания точки до получения её типа геометрии.

### Шаг 1: Создать объект Point
```csharp
Point point = new Point(40.7128, -74.006);
```
*Pro tip:* Конструктор `Point` ожидает **latitude** первым и **longitude** вторым, что соответствует традиционному порядку GIS.

### Шаг 2: Получить тип геометрии
```csharp
GeometryType geometryType = point.GeometryType;
```
Здесь мы обращаемся к свойству `GeometryType`, которое возвращает значение перечисления `GeometryType.Point`.

### Шаг 3: Отобразить тип геометрии
```csharp
Console.WriteLine(geometryType); // Point
```
Вывод в консоль подтверждает, что объект действительно **point**, и вы успешно **получили тип геометрии** из него.
