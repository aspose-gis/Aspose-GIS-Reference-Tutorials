# Как добавить точки и перебрать их в геометрии

## Введение

Если вы работаете с GIS‑данными в среде .NET, одной из первых задач будет **добавление точек** в геометрию и последующая работа с этими точками. Aspose.GIS for .NET предоставляет чистый объектно‑ориентированный API, который делает этот процесс простым. В этом руководстве мы пройдем процесс создания `LineString`, добавления в него точек и перебора этих точек, чтобы вы могли извлекать координаты или выполнять дальнейший анализ.

## Быстрые ответы
- **Какой основной класс для коллекций точек?** `LineString`
- **Как добавить точку?** Использовать `AddPoint(longitude, latitude)`
- **Можно ли перебрать точки с помощью foreach?** Да, `LineString` реализует `IEnumerable<IPoint>`
- **Требования?** .NET 6+ (или .NET Core 3.1/Framework 4.6+) и библиотека Aspose.GIS for .NET
- **Типичный сценарий использования?** Построение маршрутов, визуализация треков или предобработка данных для пространственного анализа

## Что означает «добавление точек» в GIS?

Добавление точек — это вставка отдельных пар координат (долгота, широта) в геометрический контейнер, такой как `LineString`, `Polygon` или `MultiPoint`. Каждая точка становится вершиной, определяющей форму или путь, который вы моделируете.

## Почему добавлять точки с Aspose.GIS?

- **Строгая типобезопасность** – объекты геометрии имеют строгие типы, что снижает количество ошибок во время выполнения.  
- **Кросс‑платформенность** – работает на .NET Framework, .NET Core и .NET 5/6+.  
- **Богатый API** – встроенный перебор, пространственные операции и поддержка форматов (Shapefile, GeoJSON и др.).

- Visual Studio 2022 (или любой IDE для C#)

## Как добавить точки в геометрию?

### Шаг 1: Создать объект `LineString`  

`LineString` представляет последовательность соединённых точек (полилиния). Сначала создайте экземпляр объекта:

```csharp
LineString line = new LineString();
```

### Шаг 2: Добавить точки в `LineString`  

Используйте метод `AddPoint` для вставки каждой пары координат. Это и есть основной способ **добавления точек** в вашу геометрию:

```csharp
line.AddPoint(78.65, -32.65);
line.AddPoint(-98.65, 12.65);
```

Вы можете вызывать `AddPoint` столько раз, сколько потребуется; каждый вызов добавляет новую вершину к линии.

### Шаг 3: Перебрать точки  

После того как точки добавлены, их можно пройти в цикле `foreach`. `LineString` реализует `IEnumerable<IPoint>`, что делает перебор простым и интуитивным:

```csharp
foreach (IPoint point in line)
{
    Console.WriteLine(point.X + "," + point.Y);
}
```

Цикл выводит значения X (долгота) и Y (широта) каждой точки в консоль, позволяя убедиться, что точки были добавлены корректно.

## Распространённые сценарии использования

- **Планирование маршрутов** – построение пути из GPS‑логов и последующий анализ расстояний между контрольными точками.  
- **Валидация данных** – перебор точек для проверки, что они находятся в ожидаемых границах (например, внутри границ страны).  
- **Визуализация** – экспорт `LineString` в GeoJSON или Shapefile для использования в картографических инструментах.
