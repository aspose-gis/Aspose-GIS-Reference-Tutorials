# Создание полигональной геометрии с отверстием с помощью Aspose.GIS

## Введение
В этом руководстве вы **создадите полигон с отверстием** с помощью Aspose.GIS для .NET. Независимо от того, разрабатываете ли вы картографическое приложение, выполняете пространственный анализ или готовите данные для GIS‑служб, умение встраивать отверстие в полигон является необходимым. Мы пройдем весь процесс шаг за шагом, от настройки окружения до получения окончательного объекта полигона.

## Быстрые ответы
- **Что означает «создать полигон с отверстием»?** Это значит построить полигон, содержащий одно или несколько внутренних колец (отверстий), которые исключаются из площади.
- **Какая библиотека это делает?** Aspose.GIS для .NET полностью поддерживает внешние и внутренние кольца.
- **Нужна ли лицензия?** Для разработки подходит бесплатная пробная версия; для продакшна требуется коммерческая лицензия.
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.
- **Сколько времени это займет?** Обычно менее 10 минут на реализацию и тестирование.

## Что такое «создать полигон с отверстием»?
Создание полигона с отверстием подразумевает определение **внешнего кольца**, которое очерчивает внешнюю границу, и одного или нескольких **внутренних колец**, вырезающих пустые пространства. Внутренние кольца часто называют *отверстиями*, поскольку они представляют области, не входящие в поверхность полигона.

## Почему создавать отверстие в полигоне с помощью Aspose.GIS?
- **Точное пространственное моделирование:** Реальные объекты, такие как озёра внутри земельных участков или дворы внутри зданий, требуют отверстий.
- **Совместимость:** Форматы Shapefile, GeoJSON и GML нативно поддерживают внутренние кольца; Aspose.GIS сохраняет их.
- **Производительность:** Библиотека управляет валидностью геометрии, поэтому вам не нужно писать собственный код проверки.

Теперь перейдём к созданию полигона с отверстием с помощью Aspose.GIS для .NET.

## Шаг 1: Создание объекта Polygon
Начинаем с создания пустого объекта `Polygon`, который позже будет содержать как внешнее, так и внутреннее кольца.

```csharp
Polygon polygon = new Polygon();
```

## Шаг 2: Определение внешнего кольца
Внешнее кольцо задаёт внешнюю границу полигона. Добавляйте точки по часовой стрелке, чтобы образовать замкнутую форму.

```csharp
LinearRing ring = new LinearRing();
ring.AddPoint(50.02, 36.22);
ring.AddPoint(49.99, 36.26);
ring.AddPoint(49.97, 36.23);
ring.AddPoint(49.98, 36.17);
ring.AddPoint(50.02, 36.22);
```

## Шаг 3: Определение внутреннего кольца (отверстия)
Далее создаём внутреннее кольцо — это **отверстие**, которое будет исключено из площади полигона. Точки обычно добавляются против часовой стрелки, но Aspose.GIS автоматически обрабатывает ориентацию.

```csharp
LinearRing hole = new LinearRing();
hole.AddPoint(50.00, 36.22);
hole.AddPoint(49.99, 36.20);
hole.AddPoint(49.98, 36.23);
hole.AddPoint(50.00, 36.24);
hole.AddPoint(50.00, 36.22);
```

## Шаг 4: Присвоение внешнего кольца и добавление внутреннего кольца к полигону
Наконец, привязываем внешнее кольцо к полигону, а затем добавляем внутреннее кольцо (отверстие). Метод `AddInteriorRing` можно вызывать несколько раз, если требуется несколько отверстий.

```csharp
polygon.ExteriorRing = ring;
polygon.AddInteriorRing(hole);
```

## Распространённые проблемы и решения
| Проблема | Причина | Решение |
|----------|----------|----------|
| Отверстие не отображается в GIS‑просмотрщике | Ориентация внутреннего кольца обратная | Убедитесь, что точки добавлены в противоположном направлении внешнего кольца (против часовой стрелки). |
| Ошибка «Polygon invalid» | Кольца не замкнуты (первая ≠ последняя точка) | Повторите первую точку в конце каждой цепочки (как показано выше). |
| Пустая геометрия | Не назначен `ExteriorRing` перед добавлением внутренних колец | Сначала задайте `polygon.ExteriorRing`, затем вызывайте `AddInteriorRing`. |
