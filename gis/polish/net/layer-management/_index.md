## Najczęściej zadawane pytania

**Q:** Jak mogę programowo **create new shapefile** pliki?  
**A:** Użyj klasy `Shapefile` w Aspose.GIS, określ typ geometrii, dodaj pola, a następnie zapisz obiekty. Powiązany samouczek „Create New Shapefile” pokazuje dokładny kod.

**Q:** Jaka jest różnica między File GDB a shapefile?  
**A:** File Geodatabase (File GDB) przechowuje dane w strukturze opartej na folderach z bogatszym wsparciem schematu, podczas gdy shapefile jest prostym, szeroko kompatybilnym formatem składającym się z kilku plików o stałej długości.

**Q:** Jak mogę efektywnie **filter features by attribute**?  
**A:** Zbuduj obiekt `Query` z klauzulą WHERE (np. `SELECT * FROM layer WHERE Population > 1000`). Przewodnik „Filter Features by Attribute” przeprowadzi Cię przez API.

**Q:** Czy mogę skonwertować shapefile do GeoJSON bez plików pośrednich?  
**A:** Tak — załaduj shapefile do `FeatureCollection` i bezpośrednio wywołaj `ExportToGeoJson()`. Zobacz samouczek „Extract Features to GeoJSON” po pełny przykład.

**Q:** Czy potrzebuję specjalnej licencji, aby uruchomić te przykłady lokalnie?  
**A:** Darmowa licencja ewaluacyjna działa w celach rozwoju i testowania. Do wdrożeń produkcyjnych należy zakupić licencję komercyjną od Aspose.
