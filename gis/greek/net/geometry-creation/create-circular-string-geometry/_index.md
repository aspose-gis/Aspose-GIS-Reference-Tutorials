# Δημιουργία Διανυσματικού Στρώματος και Γεωμετρίας Circular String με Aspose.GIS για .NET

## Εισαγωγή
Αν δημιουργείτε μια GIS εφαρμογή στην πλατφόρμα .NET, το πρώτο βήμα είναι συχνά **to create vector layer** αντικείμενα που αποθηκεύουν τα χωρικά σας χαρακτηριστικά. Το Aspose.GIS για .NET κάνει αυτή τη διαδικασία απλή και σας επιτρέπει να εμπλουτίσετε αυτά τα στρώματα με προχωρημένες γεωμετρίες όπως τα circular strings. Σε αυτό το tutorial θα μάθετε ακριβώς πώς να **create vector layer**, **add circular string** γεωμετρία, και να αποθηκεύσετε το αποτέλεσμα ως Shapefile — όλα με καθαρό, έτοιμο για παραγωγή κώδικα C#.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “create vector layer”;** Δημιουργεί ένα νέο κοντέινερ (στρώμα) που μπορεί να περιέχει χωρικά χαρακτηριστικά όπως σημεία, γραμμές ή πολύγωνα.  
- **Ποια κλάση αντιπροσωπεύει ένα circular string;** `CircularString` από `Aspose.Gis.Geometries`.  
- **Μπορώ να αποθηκεύσω το στρώμα ως Shapefile;** Ναι – χρησιμοποιήστε `Drivers.Shapefile` κατά τη δημιουργία του στρώματος.  
- **Χρειάζεται άδεια για ανάπτυξη;** Μια προσωρινή άδεια λειτουργεί για αξιολόγηση· απαιτείται πλήρης άδεια για παραγωγή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι είναι το “create vector layer”;
Ένα διανυσματικό στρώμα είναι μια λογική ομαδοποίηση διανυσματικών χαρακτηριστικών (σημεία, γραμμές, πολύγωνα) που αποθηκεύονται σε μία πηγή δεδομένων. Στο Aspose.GIS δημιουργείτε ένα στρώμα καλώντας `VectorLayer.Create`, περνώντας τη διαδρομή του αρχείου προορισμού και τον επιθυμητό οδηγό (π.χ., Shapefile). Μόλις το στρώμα υπάρχει, μπορείτε να προσθέσετε χαρακτηριστικά, να αναθέσετε γεωμετρίες και να εκτελέσετε χωρικές λειτουργίες.

## Γιατί να προσθέσετε ένα circular string;
Τα circular strings είναι ένας τύπος **γραμμικής γεωμετρίας** που προσεγγίζει τόξα χρησιμοποιώντας μια ακολουθία σημείων. Είναι χρήσιμα για την αναπαράσταση καμπυλωτών δρόμων, στροφών ποταμών ή οποιουδήποτε χαρακτηριστικού όπου απαιτείται ομαλή καμπύλη χωρίς την ανάγκη πολλών μικρών τμημάτων γραμμής.

```csharp
using Aspose.Gis;
using Aspose.Gis.Geometries;

```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ορισμός της διαδρομής εξόδου αρχείου
Ορίστε την τοποθεσία όπου θα γραφτεί το Shapefile.

```csharp
string path = "Your Document Directory" + "CreateCircularString_out.shp";
```

Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή φακέλου στο σύστημά σας.

### Βήμα 2: **Create vector layer**
Ανοίξτε ένα `VectorLayer` χρησιμοποιώντας τη μέθοδο `Create`. Αυτό αποτελεί τον πυρήνα της λειτουργίας **create vector layer**.

```csharp
using (VectorLayer layer = VectorLayer.Create(path, Drivers.Shapefile))
{
```

### Βήμα 3: Δημιουργία νέου feature
Ένα feature αντιπροσωπεύει μια μεμονωμένη χωρική εγγραφή μέσα στο στρώμα.

```csharp
    var feature = layer.ConstructFeature();
```

### Βήμα 4: Δημιουργία της γεωμετρίας circular string
Προσθέστε τα σημεία που ορίζουν το καμπυλωτό σχήμα. Η ακολουθία των σημείων δημιουργεί ένα τόξο που ξεκινά και τελειώνει στην ίδια θέση, σχηματίζοντας ένα κλειστό circular string.

```csharp
    var circularString = new CircularString();
    circularString.AddPoint(0, 0);
    circularString.AddPoint(1, 1);
    circularString.AddPoint(2, 0);
    circularString.AddPoint(1, -1);
    circularString.AddPoint(0, 0);
```

### Βήμα 5: Ανάθεση γεωμετρίας και προσθήκη του feature στο στρώμα
Συνδέστε τη γεωμετρία με το feature και αποθηκεύστε το στο στρώμα.

```csharp
    feature.Geometry = circularString;
    layer.Add(feature);
}
```

Όταν το μπλοκ `using` ολοκληρωθεί, το στρώμα αποστέλλεται αυτόματα στο Shapefile στον δίσκο.

## FAQ (Γρήγορη‑Αναφορά)

**Q:** Πώς μπορώ προγραμματιστικά **create vector layer**;  
**A:** Καλέστε `VectorLayer.Create(path, Drivers.Shapefile)` (ή άλλο driver) μέσα σε ένα μπλοκ `using`.

**Q:** Ποια μέθοδος προσθέτει σημεία σε ένα circular string;  
**A:** Χρησιμοποιήστε `circularString.AddPoint(x, y)` για κάθε συντεταγμένη.

**Q:** Μπορώ να αποθηκεύσω πολλαπλές γεωμετρίες στο ίδιο στρώμα;  
**A:** Ναι, δημιουργήστε ένα νέο feature για κάθε γεωμετρία και προσθέστε το με `layer.Add(feature)`.

**Q:** Τι πρέπει να κάνω αν το Shapefile δεν δημιουργηθεί;  
**A:** Επαληθεύστε ότι ο φάκελος εξόδου υπάρχει, έχετε δικαιώματα εγγραφής, και ότι ο οδηγός (`Drivers.Shapefile`) έχει αναφερθεί σωστά.

**Q:** Απαιτείται άδεια για την έκδοση αξιολόγησης;  
**A:** Μια προσωρινή άδεια είναι επαρκής για ανάπτυξη και δοκιμές· πλήρης άδεια απαιτείται για παραγωγικές εγκαταστάσεις.
