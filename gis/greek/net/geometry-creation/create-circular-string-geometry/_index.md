# Δημιουργία διανυσματικού στρώματος και γεωμετρίας κυκλικής συμβολοσειράς με Aspose.GIS για .NET

## Εισαγωγή
Αν δημιουργείτε μια εφαρμογή GIS στην πλατφόρμα .NET, το πρώτο βήμα είναι συχνά **να δημιουργήσετε διανυσματικό στρώμα** αντικείμενα που αποθηκεύουν τα χωρικά σας χαρακτηριστικά. Το Aspose.GIS για .NET καθιστά αυτή τη διαδικασία απλή και σας επιτρέπει να εμπλουτίσετε αυτά τα στρώματα με προχωρημένες γεωμετρίες όπως οι κυκλικές συμβολοσειρές. Σε αυτό το σεμινάριο θα μάθετε ακριβώς πώς να δημιουργήσετε ένα διανυσματικό στρώμα, να προσθέσετε μια γεωμετρία κυκλικής συμβολοσειράς και να αποθηκεύσετε το αποτέλεσμα ως Shapefile — όλα με καθαρό, έτοιμο για παραγωγή κώδικα C#.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει το “create vector layer”;** Δημιουργεί ένα νέο κοντέινερ (στρώμα) που μπορεί να περιέχει χωρικά χαρακτηριστικά όπως σημεία, γραμμές ή πολύγωνα.  
- **Ποια κλάση αντιπροσωπεύει μια κυκλική συμβολοσειρά;** `CircularString` από `Aspose.Gis.Geometries`.  
- **Μπορώ να αποθηκεύσω το στρώμα ως Shapefile;** Ναι – χρησιμοποιήστε `Drivers.Shapefile` κατά τη δημιουργία του στρώματος.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια προσωρινή άδεια λειτουργεί για αξιολόγηση· απαιτείται πλήρης άδεια για παραγωγή.  
- **Ποιες εκδόσεις του .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι είναι το “create vector layer”;
Ένα διανυσματικό στρώμα είναι μια λογική ομαδοποίηση διανυσματικών χαρακτηριστικών (σημείων, γραμμών, πολυγώνων) που αποθηκεύονται σε μία ενιαία πηγή δεδομένων. Στο Aspose.GIS δημιουργείτε ένα στρώμα καλώντας `VectorLayer.Create`, περνώντας τη διαδρομή του αρχείου προορισμού και τον επιθυμητό οδηγό (π.χ., Shapefile). Μόλις το στρώμα υπάρχει, μπορείτε να προσθέσετε χαρακτηριστικά, να αναθέσετε γεωμετρίες και να εκτελέσετε χωρικές λειτουργίες.

## Γιατί να προσθέσετε μια κυκλική συμβολοσειρά;
Οι κυκλικές συμβολοσειρές είναι ένας τύπος **γραμμικής γεωμετρίας** που προσεγγίζει τόξα χρησιμοποιώντας μια ακολουθία σημείων. Είναι χρήσιμες για την αναπαράσταση καμπυλωτών δρόμων, στροφών ποταμών ή οποιουδήποτε χαρακτηριστικού όπου απαιτείται ομαλή καμπύλη χωρίς την ανάγκη πολλών μικρών τμημάτων γραμμής.

## Προαπαιτούμενα
- **.NET Framework ή .NET Core** εγκατεστημένο στον υπολογιστή σας.  
- **Βιβλιοθήκη Aspose.GIS για .NET** – κατεβάστε την από την επίσημη ιστοσελίδα **[εδώ](https://releases.aspose.com/gis/net/)**.  
- Ένα IDE όπως το **Visual Studio** ή το **JetBrains Rider**.

## Εισαγωγή Ονομάτων Χώρου
Προσθέστε τα απαιτούμενα ονόματα χώρου στο αρχείο C# σας:

```csharp
using Aspose.Gis;
using Aspose.Gis.Geometries;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

## Οδηγός Βήμα‑βήμα

### Βήμα 1: Ορισμός διαδρομής εξόδου αρχείου
Ορίστε τη θέση όπου θα γραφτεί το Shapefile.

```csharp
string path = "Your Document Directory" + "CreateCircularString_out.shp";
```

Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή φακέλου στο σύστημά σας.

### Βήμα 2: **Δημιουργία διανυσματικού στρώματος**
Ανοίξτε ένα `VectorLayer` χρησιμοποιώντας τη μέθοδο `Create`. Αυτό αποτελεί τον πυρήνα της λειτουργίας **create vector layer**.

```csharp
using (VectorLayer layer = VectorLayer.Create(path, Drivers.Shapefile))
{
```

### Βήμα 3: Δημιουργία νέου χαρακτηριστικού
Ένα χαρακτηριστικό αντιπροσωπεύει μία μοναδική χωρική εγγραφή μέσα στο στρώμα.

```csharp
    var feature = layer.ConstructFeature();
```

### Βήμα 4: Κατασκευή γεωμετρίας κυκλικής συμβολοσειράς
Προσθέστε τα σημεία που ορίζουν το καμπυλωτό σχήμα. Η ακολουθία των σημείων δημιουργεί ένα τόξο που ξεκινά και τελειώνει στην ίδια θέση, σχηματίζοντας μια κλειστή κυκλική συμβολοσειρά.

```csharp
    var circularString = new CircularString();
    circularString.AddPoint(0, 0);
    circularString.AddPoint(1, 1);
    circularString.AddPoint(2, 0);
    circularString.AddPoint(1, -1);
    circularString.AddPoint(0, 0);
```

### Βήμα 5: Ανάθεση γεωμετρίας και προσθήκη του χαρακτηριστικού στο στρώμα
Συνδέστε τη γεωμετρία με το χαρακτηριστικό και αποθηκεύστε το στο στρώμα.

```csharp
    feature.Geometry = circularString;
    layer.Add(feature);
}
```

Όταν ολοκληρωθεί το μπλοκ `using`, το στρώμα αποθηκεύεται αυτόματα στο Shapefile στον δίσκο.
