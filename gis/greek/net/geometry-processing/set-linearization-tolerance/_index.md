# Δημιουργία Στρώματος Διανυσματικών Δεδομένων και Ορισμός Ανοχής Γραμμικοποίησης χρησιμοποιώντας το Aspose.GIS για .NET

## Εισαγωγή
Αν χρειάζεστε να **create vector layer** αρχεία, να ελέγχετε την ακρίβεια των καμπυλών και να εξάγετε το αποτέλεσμα ως έγγραφο GeoJSON, το Aspose.GIS για .NET το κάνει απλό. Σε αυτό το tutorial θα μάθετε πώς να διαμορφώσετε τις επιλογές GeoJSON, να ορίσετε την ανοχή γραμμικοποίησης και να **add feature to layer** αντικείμενα—όλα ενώ διατηρείτε τον κώδικα καθαρό και έτοιμο για παραγωγή.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “create vector layer”;** Δημιουργεί ένα νέο GIS διανυσματικό σύνολο δεδομένων (π.χ., αρχείο GeoJSON) που μπορεί να αποθηκεύσει γεωμετρίες και χαρακτηριστικά.  
- **Πώς να ορίσετε την ανοχή;** Χρησιμοποιήστε την ιδιότητα `LinearizationTolerance` του `GeoJsonOptions`.  
- **Μπορώ να εξάγω αρχείο GeoJSON;** Ναι—απλώς δημιουργήστε ένα `VectorLayer` με τον οδηγό `Drivers.GeoJson`.  
- **Χρειάζομαι άδεια;** Μια έγκυρη άδεια Aspose.GIS ξεκλειδώνει όλες τις λειτουργίες· μια προσωρινή άδεια λειτουργεί για αξιολόγηση.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι σημαίνει “create vector layer”;
Η δημιουργία ενός διανυσματικού στρώματος σημαίνει την αρχικοποίηση ενός νέου GIS container (όπως αρχείο GeoJSON) που μπορεί να περιέχει γεωμετρικά χαρακτηριστικά όπως σημεία, γραμμές και πολύγωνα. Αυτό το στρώμα γίνεται ο προορισμός για οποιαδήποτε γεωμετρία δημιουργήσετε και οποιαδήποτε χαρακτηριστικά προσθέσετε.

## Γιατί να διαμορφώσετε τις επιλογές GeoJSON;
Η διαμόρφωση των επιλογών GeoJSON—ιδιαίτερα η **linearization tolerance**—εξασφαλίζει ότι οι καμπύλες γεωμετρίες (π.χ., `CircularString`) προσεγγίζονται με ακρίβεια που ικανοποιεί τις απαιτήσεις του εφαρμογής σας. Αυτό το βήμα είναι κρίσιμο όταν αργότερα **export GeoJSON file** για χρήση σε web χάρτες ή χωρικές αναλύσεις.

## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

1. **Visual Studio** εγκατεστημένο (οποιαδήποτε πρόσφατη έκδοση).  
2. **Aspose.GIS license** (ή προσωρινό κλειδί αξιολόγησης).  
3. **Aspose.GIS for .NET** βιβλιοθήκη κατεβασμένη και αναφορά στο πρότζεκτ σας.  
4. Βασική εξοικείωση με **C#**.

## Εισαγωγή Ονοματοχώρων
Πρώτα, εισάγετε τους απαιτούμενους ονοματοχώρους ώστε ο μεταγλωττιστής να γνωρίζει πού βρίσκονται οι κλάσεις GIS:

```csharp
using Aspose.Gis;
using Aspose.Gis.Formats.GeoJson;
using Aspose.Gis.Geometries;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Διαμόρφωση Επιλογών GeoJSON (πώς να ορίσετε την ανοχή)
Θα δημιουργήσουμε ένα αντικείμενο `GeoJsonOptions` και θα πούμε στο Aspose.GIS πόσο κοντά πρέπει να είναι η γραμμικοποιημένη γεωμετρία στην αρχική καμπύλη.

```csharp
var options = new GeoJsonOptions
{
    // linearized geometry must be within 1e-4 from curve geometry
    LinearizationTolerance = 1e-4,
};
```

### Βήμα 2: Ορισμός Διαδρομής Εξόδου (πώς να εξάγετε GeoJSON)
Καθορίστε πού θα αποθηκευτεί το παραγόμενο αρχείο. Αντικαταστήστε το placeholder με το πραγματικό σας φάκελο.

```csharp
string path = "Your Document Directory" + "SpecifyLinearizationTolerance_out.json";
```

### Βήμα 3: **Create Vector Layer** με τις διαμορφωμένες επιλογές
Τώρα δημιουργούμε πραγματικά **create vector layer** χρησιμοποιώντας τη μέθοδο `VectorLayer.Create`. Το μπλοκ `using` εγγυάται τη σωστή διαχείριση των πόρων.

```csharp
using (VectorLayer layer = VectorLayer.Create(path, Drivers.GeoJson, options))
{
    // Your code here
}
```

### Βήμα 4: Δημιουργία Γεωμετρίας (π.χ., μια κυκλική ακολουθία)
Εδώ κατασκευάζουμε ένα δείγμα γεωμετρίας—ένα `CircularString`. Μπορείτε να το αντικαταστήσετε με οποιοδήποτε έγκυρο WKT.

```csharp
var curveGeometry = Geometry.FromText("CircularString (0 0, 1 1, 2 0)");
```

### Βήμα 5: **Add Feature to Layer** και αποθήκευση
Τέλος, δημιουργούμε ένα feature, αναθέτουμε τη γεωμετρία και το προσθέτουμε στο στρώμα. Αυτό είναι το κύριο της λειτουργίας **add feature to layer**.

```csharp
var feature = layer.ConstructFeature();
feature.Geometry = curveGeometry;
layer.Add(feature);
```

Όταν το μπλοκ `using` ολοκληρωθεί, το στρώμα γράφεται αυτόματα στο αρχείο που ορίστηκε στο `path`, παρέχοντάς σας ένα έτοιμο προς χρήση αρχείο GeoJSON.

## Συχνά Προβλήματα & Συμβουλές
- **Incorrect path** – Βεβαιωθείτε ότι ο φάκελος υπάρχει και έχετε δικαιώματα εγγραφής.  
- **Tolerance too low** – Ο ορισμός `LinearizationTolerance` σε πολύ μικρή τιμή μπορεί να αυξήσει δραματικά το μέγεθος του αρχείου. Ρυθμίστε το ανάλογα με τις ανάγκες ακρίβειας.  
- **License errors** – Αν εμφανιστούν προειδοποιήσεις άδειας, ελέγξτε ότι το αρχείο άδειας έχει φορτωθεί σωστά πριν από οποιαδήποτε κλήση Aspose.GIS.
