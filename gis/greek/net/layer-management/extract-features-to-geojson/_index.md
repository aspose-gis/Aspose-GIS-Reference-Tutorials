## Τι είναι η “μετατροπή shapefile σε geojson”;
Η μετατροπή ενός Shapefile σε GeoJSON σημαίνει ανάγνωση διανυσματικών χαρακτηριστικών από την κλασική μορφή ESRI Shapefile και εγγραφή τους ως ένα σύγχρονο, φιλικό προς το web έγγραφο GeoJSON. Το GeoJSON χρησιμοποιείται ευρέως σε βιβλιοθήκες χαρτογράφησης JavaScript (Leaflet, Mapbox GL) και APIs, καθιστώντας τη μετατροπή μια συχνή εργασία στην ανάπτυξη GIS.

## Γιατί να χρησιμοποιήσετε το Aspose.GIS για αυτή τη μετατροπή;
Το Aspose.GIS αφαιρεί τις λεπτομέρειες χαμηλού επιπέδου των μορφών αρχείων, σας επιτρέπει να αντιγράφετε πίνακες χαρακτηριστικών χωρίς κόπο, και παρέχει ένα καθαρό, αντικειμενοστραφές API που λειτουργεί σε .NET Framework, .NET Core και .NET 5/6. Αυτό σημαίνει ότι μπορείτε να εστιάσετε στη λογική της επιχείρησης αντί να επεξεργάζεστε δυαδικά αρχεία.

## Πώς να Μετατρέψετε Shapefile σε GeoJSON
Παρακάτω είναι η πλήρης ροή εργασίας χωρισμένη σε σαφή βήματα. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση ακολουθούμενη από το ακριβές μπλοκ κώδικα που πρέπει να αντιγράψετε στο έργο σας.

### Βήμα 1: Άνοιγμα Εισόδου Shapefile
```csharp
using (VectorLayer inputLayer = VectorLayer.Open(dataDir + "InputShapeFile.shp", Drivers.Shapefile))
{
    // Your code for processing the input shapefile goes here
}
```
Ανοίξτε το Shapefile εισόδου χρησιμοποιώντας τη μέθοδο `VectorLayer.Open`. Αυτό σας δίνει μια συλλογή `Feature` αντικειμένων που μπορεί να επαναληφθεί.

### Βήμα 2: Δημιουργία Εξόδου GeoJSON
```csharp
using (VectorLayer outputLayer = VectorLayer.Create(dataDir + "ExtractFeaturesFromShapeFileToGeoJSON_out.json", Drivers.GeoJson))
{
    // Your code for creating the output GeoJSON goes here
}
```
Δημιουργήστε το αρχείο εξόδου με τη μέθοδο `VectorLayer.Create`, καθορίζοντας τον οδηγό `Drivers.GeoJson`.

### Βήμα 3: Αντιγραφή Χαρακτηριστικών μεταξύ Επιπέδων
```csharp
outputLayer.CopyAttributes(inputLayer);
```
Αυτή η μοναδική γραμμή αντιγράφει το σχήμα των χαρακτηριστικών (ονόματα πεδίων, τύπους) από το πηγαίο Shapefile στο νέο επίπεδο GeoJSON—ακριβώς όπως περιγράφει η δευτερεύουσα λέξη-κλειδί *copy attributes between layers*.

### Βήμα 4: Επεξεργασία Χαρακτηριστικών
```csharp
foreach (Feature inputFeature in inputLayer)
{
    // Your code for processing each input feature goes here
}
```
Επαναλάβετε κάθε χαρακτηριστικό στο Shapefile ώστε να μπορείτε να εφαρμόσετε οποιαδήποτε προσαρμοσμένη λογική πριν το γράψετε.

### Βήμα 5: Φιλτράρισμα Χαρακτηριστικών κατά Ημερομηνία
```csharp
DateTime? date = inputFeature.GetValue<DateTime?>("dob");
if (date == null || date < new DateTime(1982, 1, 1))
{
    continue;
}
```
Σε αυτό το παράδειγμα παραλείπουμε εγγραφές των οποίων το πεδίο `dob` (ημερομηνία γέννησης) λείπει ή είναι νωρίτερα από 1 Ιαν 1982. Μπορείτε να προσαρμόσετε το φίλτρο ώστε να ταιριάζει στις δικές σας απαιτήσεις δεδομένων.

### Βήμα 6: Δημιουργία Νέου Χαρακτηριστικού (C# Δημιουργία Αρχείου GeoJSON)
```csharp
Feature outputFeature = outputLayer.ConstructFeature();
outputFeature.Geometry = inputFeature.Geometry;
outputFeature.CopyValues(inputFeature);
outputLayer.Add(outputFeature);
```
Εδώ δημιουργούμε ένα νέο `Feature` για το επίπεδο GeoJSON, αντιγράφουμε τη γεωμετρία και τις τιμές χαρακτηριστικών, και το προσθέτουμε στην έξοδο. Αυτό είναι ο πυρήνας του *c# generate geojson file*.
