# Πώς να Κόψετε Χαρακτηριστικά Στρώματος

## Εισαγωγή
Σε αυτό το tutorial θα μάθετε **πώς να κόψετε χαρακτηριστικά στρώματος** χρησιμοποιώντας το Aspose.GIS για .NET, μια ισχυρή προσέγγιση που σας επιτρέπει να **κόψετε raster με πολύγωνο**, **εξάγετε το μέγεθος κελιού raster**, και **ανακτήσετε το εύρος raster** για ακριβή γεωχωρική ανάλυση. Είτε προετοιμάζετε δεδομένα για web χάρτη, κόβετε δορυφορικές εικόνες, είτε απομονώνετε μια περιοχή ενδιαφέροντος, αυτός ο οδηγός βήμα‑βήμα σας δείχνει ακριβώς πώς να ολοκληρώσετε την εργασία.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “crop layer”;** Κόβει ένα raster ή vector στρώμα στα όρια μιας δοθείσας γεωμετρίας.  
- **Ποια γεωμετρία χρησιμοποιείται σε αυτό το παράδειγμα;** Ένα πολύγωνο ορισμένο σε μορφή WKT.  
- **Μπορώ να εξάγω το μέγεθος κελιού μετά την κοπή;** Ναι – η ιδιότητα `CellSize` παρέχει αυτή την πληροφορία.  
- **Χρειάζεται άδεια για την εκτέλεση του κώδικα;** Μια προσωρινή άδεια λειτουργεί για αξιολόγηση· απαιτείται πλήρης άδεια για παραγωγή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι είναι το “crop layer”;
Η κοπή ενός στρώματος σημαίνει περιορισμό του συνόλου δεδομένων σε συγκεκριμένη γεωγραφική περιοχή, απορρίπτοντας ό,τι βρίσκεται εκτός του καθορισμένου σχήματος. Αυτό μειώνει το μέγεθος του αρχείου, επιταχύνει την επεξεργασία και εστιάζει την ανάλυση στην περιοχή που σας ενδιαφέρει.

## Γιατί να χρησιμοποιήσετε το Aspose.GIS για κοπή;
- **Υψηλής απόδοσης διαχείριση raster** – ιδανική για μεγάλα αρχεία GeoTiff.  
- **Απλό API** – κλήση `Crop` μίας γραμμής με οποιαδήποτε γεωμετρία.  
- **Πλήρης συμβατότητα με .NET** – λειτουργεί σε desktop, server και cloud εφαρμογές.  
- **Ακριβής διαχείριση αναφοράς χώρου** – διατηρεί αυτόματα τις πληροφορίες CRS.

Τώρα, ας προχωρήσουμε στον οδηγό βήμα‑βήμα.

## Εισαγωγή Namespaces
Ξεκινήστε εισάγοντας τα απαραίτητα namespaces για να αξιοποιήσετε πλήρως τη δύναμη του Aspose.GIS:

```csharp
using System;
using System.IO;
using Aspose.Gis;
using Aspose.Gis.Geometries;
```

## Βήμα 1: Άνοιγμα και Κοπή του Στρώματος (crop raster with polygon)
Ξεκινήστε ανοίγοντας το στρώμα GeoTiff και κόβοντάς το βάσει ενός ορισμένου πολυγώνου. Αυτό εξασφαλίζει ότι τα γεωχωρικά σας δεδομένα περιορίζονται σε συγκεκριμένη περιοχή ενδιαφέροντος.

```csharp
using (var layer = Drivers.GeoTiff.OpenLayer(Path.Combine(filesPath, "geodetic_world.tif")))
using (var warped = layer.Crop(Geometry.FromText("POLYGON ((-160 0, 0 60, 160 0, 0 -160, -160 0))")))
{
```

## Βήμα 2: Ανάκτηση Πληροφοριών Raster (extract raster cell size & retrieve raster extent)
Μόλις το στρώμα κοπεί, εξάγετε βασικές πληροφορίες για τα raster δεδομένα, όπως το μέγεθος κελιού, το σύστημα αναφοράς χώρου και τα όρια.

```csharp
// read and print raster
var cellSize = warped.CellSize;
var extent = warped.GetExtent();
var spatialRefSys = warped.SpatialReferenceSystem;
var code = spatialRefSys == null ? "'no srs'" : spatialRefSys.EpsgCode.ToString();
var bounds = warped.Bounds;
```

## Βήμα 3: Εμφάνιση Πληροφοριών
Εκτυπώστε τις εξαγόμενες πληροφορίες για να κατανοήσετε την επίδραση της διαδικασίας κοπής στα γεωχωρικά σας δεδομένα.

```csharp
Console.WriteLine($"cellSize: {cellSize}");
Console.WriteLine($"source extent: {layer.GetExtent()}");
Console.WriteLine($"target extent: {extent}");
Console.WriteLine($"spatialRefSys: {code}");
Console.WriteLine($"bounds: {bounds}");
```

Επαναλάβετε αυτά τα βήματα όσο χρειάζεται για να βελτιώσετε και να προσαρμόσετε τα γεωχωρικά σας δεδομένα ώστε να καλύψουν τις συγκεκριμένες απαιτήσεις του έργου σας.

## Συνηθισμένα Προβλήματα και Λύσεις
| Πρόβλημα | Λύση |
|-------|----------|
| **Λανθασμένος προσανατολισμός πολυγώνου** | Βεβαιωθείτε ότι το WKT πολύγωνο ακολουθεί τον κανόνα του δεξιού χεριού (αντίστροφη φορά για εξωτερικούς δακτυλίους). |
| **Απουσία αναφοράς χώρου** | Επαληθεύστε ότι το πηγαίο GeoTiff περιέχει CRS· εάν όχι, ορίστε το χειροκίνητα πριν την κοπή. |
| **Μείωση απόδοσης σε τεράστια rasters** | Χρησιμοποιήστε `layer.Crop` σε ένα υποδειγματικό αντίγραφο ή επεξεργαστείτε το raster σε πλακίδια. |
