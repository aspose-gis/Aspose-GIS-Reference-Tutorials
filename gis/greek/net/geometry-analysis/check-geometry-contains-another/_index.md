# point inside polygon c# – Έλεγχος Geometry Contains Another

## Εισαγωγή
Αν εργάζεστε σε έργα **geospatial analysis .net**, ένα από τα πιο συνηθισμένα καθήκοντα είναι να προσδιορίσετε εάν μια συγκεκριμένη τοποθεσία (ένα σημείο) βρίσκεται μέσα σε μια καθορισμένη περιοχή (ένα πολύγωνο). Σε αυτό το εκπαιδευτικό υλικό θα σας δείξουμε, βήμα‑βήμα, πώς να εκτελέσετε έναν έλεγχο **point inside polygon c#** με τη βιβλιοθήκη **Aspose.GIS .NET**. Είτε δημιουργείτε μια εφαρμογή χαρτογράφησης, μια υπηρεσία βασισμένη στην τοποθεσία, ή οποιαδήποτε λύση που απαιτεί λογική χωρικής περιεκτικότητας, τα παρακάτω αποσπάσματα κώδικα θα σας θέσουν σε λειτουργία μέσα σε λίγα λεπτά.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “point inside polygon c#”;** Είναι ένα χωρικό ερώτημα που επιστρέφει true όταν η γεωμετρία ενός σημείου βρίσκεται πλήρως μέσα σε μια γεωμετρία πολυγώνου.  
- **Ποια βιβλιοθήκη το διαχειρίζεται στο .NET;** Η Aspose.GIS for .NET παρέχει τις μεθόδους `SpatiallyContains` και `Within`.  
- **Χρειάζομαι άδεια;** Διατίθεται δωρεάν δοκιμή· απαιτείται εμπορική άδεια για χρήση σε παραγωγή.  
- **Είναι συμβατή με .NET Core / .NET 6+;** Ναι – η Aspose.GIS υποστηρίζει πλήρως σύγχρονα .NET runtime.  
- **Πόσο χρόνο απαιτεί η υλοποίηση;** Περίπου 10 λεπτά για να αντιγράψετε τον κώδικα και να εκτελέσετε το παράδειγμα.

## Τι είναι το point inside polygon c#;
Ένα τεστ *point inside polygon* ελέγχει εάν οι συντεταγμένες ενός αντικειμένου `Point` βρίσκονται εντός των ορίων ενός αντικειμένου `Polygon`. Στη C# αυτό συνήθως επιτυγχάνεται μέσω βιβλιοθηκών γεωμετρίας που υλοποιούν τους αλγόριθμους **Ray Casting** ή **Winding Number**. Η Aspose.GIS αφαιρεί αυτές τις λεπτομέρειες και προσφέρει ένα απλό API: `polygon.SpatiallyContains(point)`.

- **Πλούσιο μοντέλο γεωμετρίας** – Υποστηρίζει πολύγωνα, πολλαπλά πολύγωνα, γραμμικούς δακτυλίους και άλλα.

## Εισαγωγή Namespaces
Στο έργο C# σας, εισάγετε τα απαραίτητα namespaces για να αξιοποιήσετε τις λειτουργίες της Aspose.GIS:
```csharp
using Aspose.Gis.Geometries;
using System;

```

## Βήμα 1: Ορισμός Αντικειμένων Γεωμετρίας
Αρχικά, ορίστε τα αντικείμενα γεωμετρίας χρησιμοποιώντας τις κλάσεις της Aspose.GIS. Εδώ δημιουργούμε ένα πολύγωνο με εξωτερικό δακτύλιο και εσωτερικό δακτύλιο (τρύπα), στη συνέχεια ένα σημείο που θα ελέγξουμε για περιεκτικότητα.
```csharp
var geometry1 = new Polygon();
geometry1.ExteriorRing = new LinearRing(new[]
{
    new Point(0, 0),
    new Point(0, 4),
    new Point(4, 4),
    new Point(4, 0),
    new Point(0, 0),
});
geometry1.AddInteriorRing(new LinearRing(new[]
{
    new Point(1, 1),
    new Point(1, 3),
    new Point(3, 3),
    new Point(3, 1),
    new Point(1, 1),
}));
var geometry2 = new Point(2, 2);
```

## Βήμα 2: Έλεγχος Χωρικής Περιεκτικότητας
Στη συνέχεια, ελέγξτε εάν το πολύγωνο **geometry1** περιέχει το σημείο **geometry2**. Η μέθοδος `SpatiallyContains` επιστρέφει `false` επειδή το σημείο βρίσκεται μέσα στον εσωτερικό δακτύλιο (τη τρύπα).
```csharp
Console.WriteLine(geometry1.SpatiallyContains(geometry2)); // False
```

## Βήμα 3: Ορισμός Άλλης Γεωμετρίας
Τώρα ορίζουμε ένα δεύτερο σημείο που βρίσκεται στον εξωτερικό δακτύλιο αλλά εκτός του εσωτερικού δακτυλίου.
```csharp
var geometry3 = new Point(0.5, 0.5);
```

## Βήμα 4: Έλεγχος Χωρικής Περιεκτικότητας Ξανά
Η εκτέλεση του ίδιου ελέγχου περιεκτικότητας με το νέο σημείο επιστρέφει `true`, επιβεβαιώνοντας ότι το σημείο βρίσκεται πράγματι εντός του εξωτερικού ορίου του πολυγώνου.
```csharp
Console.WriteLine(geometry1.SpatiallyContains(geometry3)); // True
```

## Βήμα 5: Ισοδύναμη Λειτουργικότητα
Η Aspose.GIS παρέχει επίσης την αντίστροφη μέθοδο `Within`. Η παρακάτω γραμμή δείχνει ότι το `geometry3.Within(geometry1)` δίνει το ίδιο αποτέλεσμα με το `geometry1.SpatiallyContains(geometry3)`.
```csharp
Console.WriteLine(geometry3.Within(geometry1)); // True
```

| **NullReferenceException** | Τα αντικείμενα γεωμετρίας δεν έχουν αρχικοποιηθεί πριν κληθεί η `SpatiallyContains`. | Δημιουργήστε και τα δύο αντικείμενα, το πολύγωνο και το σημείο, πριν καλέσετε τις χωρικές μεθόδους. |
| **Μείωση απόδοσης σε μεγάλα σύνολα δεδομένων** | Δημιουργία αντικειμένων γεωμετρίας επανειλημμένα μέσα σε βρόχους. | Επαναχρησιμοποιήστε τις υπάρχουσες αντικειμενικές εμφανίσεις ή επεξεργαστείτε σε παρτίδες χρησιμοποιώντας `GeometryCollection`. |
