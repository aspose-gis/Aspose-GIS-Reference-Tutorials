# Πώς να δημιουργήσετε χάρτη SVG και να προσθέσετε πόλεις με Aspose.GIS για .NET

## Εισαγωγή
Καλώς ήρθατε στον συναρπαστικό κόσμο του Aspose.GIS για .NET! Σε αυτόν τον οδηγό βήμα‑βήμα, θα μάθετε **πώς να προσθέτετε πόλεις σε έναν χάρτη** και **πώς να δημιουργείτε έξοδο χάρτη SVG** που φαίνεται εξαιρετικά σε οποιαδήποτε οθόνη. Είτε δημιουργείτε έναν πίνακα ελέγχου για επιτραπέζιο υπολογιστή, είτε μια διαδικτυακή πύλη GIS, είτε μια εκτυπώσιμη αναφορά, ο ίδιος κώδικας λειτουργεί σε .NET Framework, .NET Core και .NET 5/6. Ας περάσουμε από όλη τη διαδικασία — από τον καθορισμό των διαστάσεων του χάρτη μέχρι την εξαγωγή ενός καθαρού, κλιμακώσιμου αρχείου SVG.

## Γρήγορες Απαντήσεις
- **Τι καλύπτει αυτός ο οδηγός;** Προσθήκη σημείων πόλεων σε χάρτη και εξαγωγή του αποτελέσματος ως αρχείο SVG.  
- **Ποια βιβλιοθήκη απαιτείται;** Aspose.GIS για .NET (διαθέσιμο δωρεάν δοκιμαστικό).  
- **Χρειάζομαι άδεια;** Η δοκιμαστική έκδοση λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγική χρήση.  
- **Μπορώ να το χρησιμοποιήσω σε web εφαρμογές;** Απόλυτα – ο ίδιος κώδικας εκτελείται σε ASP.NET, Blazor και άλλα .NET web frameworks.  
- **Τι μορφή εξόδου παράγεται;** Ένας χάρτης SVG υψηλής ανάλυσης που οι browsers αποδίδουν άμεσα και οι επεξεργαστές διανυσματικών γραφικών μπορούν να τον επεξεργαστούν περαιτέρω.

## Τι είναι η «δημιουργία χάρτη SVG»;
*Η δημιουργία ενός χάρτη SVG* σημαίνει τη μετατροπή γεωγραφικών δεδομένων σε αρχείο Scalable Vector Graphics που διατηρεί τη γεωμετρία, τα στυλ και την αλληλεπίδραση χωρίς να ραστεροποιεί την εικόνα. Τα αρχεία SVG παραμένουν καθαρά σε οποιοδήποτε επίπεδο ζουμ και είναι ιδανικά για οπτικοποιήσεις στο web.

## Γιατί να δημιουργήσετε χάρτη SVG με το Aspose.GIS;
Το Aspose.GIS υποστηρίζει **πάνω από 70 μορφές εισόδου και εξόδου** (συμπεριλαμβανομένων Shapefile, GeoJSON, KML και GML) και μπορεί να αποδίδει χάρτες με **υπο‑pixel ακρίβεια** διατηρώντας χαμηλή χρήση μνήμης. Η βιβλιοθήκη επεξεργάζεται **σύνολα δεδομένων εκατοντάδων σελίδων** χωρίς να φορτώνει ολόκληρο το αρχείο στη μνήμη, καθιστώντας την ιδανική για μεγάλης κλίμακας έργα GIS.

Πριν βυθιστείτε στον οδηγό, βεβαιωθείτε ότι έχετε τα παρακάτω προαπαιτούμενα:
- Aspose.GIS for .NET Library: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.GIS for .NET. Μπορείτε να τη κατεβάσετε [εδώ](https://releases.aspose.com/gis/net/).
- Αρχεία Δεδομένων: Προετοιμάστε τα απαραίτητα shapefiles και δεδομένα GeoJSON για τον οδηγό. Μπορείτε να βρείτε δείγμα δεδομένων στην τεκμηρίωση ή να χρησιμοποιήσετε τα δικά σας αρχεία.
- Περιβάλλον Ανάπτυξης: Διαθέστε ένα περιβάλλον ανάπτυξης .NET, συμπεριλαμβανομένου ενός επεξεργαστή κώδικα όπως το Visual Studio.

## Εισαγωγή Ονομάτων Χώρων
Το namespace `Aspose.Gis` παρέχει όλες τις βασικές λειτουργίες GIS, ενώ το `Aspose.Gis.Rendering` περιέχει κλάσεις ειδικές για την απόδοση.

```csharp
using Aspose.Gis;
using Aspose.Gis.Rendering;
using Aspose.Gis.Rendering.Symbolizers;
using Aspose.Gis.SpatialReferencing;
using Aspose.GIS.Examples.CSharp;
using System.Drawing;
using System.Drawing.Drawing2D;
using System.Drawing.Text;
using System.IO;
using System.Linq;
```

## Πώς ορίζω τις διαστάσεις του χάρτη;
`Map` είναι η βασική κλάση που κρατά την επιφάνεια απόδοσης και διαχειρίζεται τα επίπεδα.  
Ορίστε πρώτα το μέγεθος του καμβά του χάρτη ώστε ο renderer να γνωρίζει πόσο χώρο να διαθέσει. Δημιουργείτε ένα αντικείμενο `Map`, περνάτε το πλάτος και το ύψος σε εικονοστοιχεία (pixels) και προαιρετικά ορίζετε ένα χρώμα φόντου. Αυτό το βήμα ελέγχει την αναλογία διαστάσεων του τελικού SVG, το μέγεθος του αρχείου και την οπτική καθαρότητα σε διαφορετικές συσκευές.

```csharp
string dataDir = "Your Document Directory";
using (var map = new Map(800, 476))
{
    // Additional code for map setup can be added here.
}
```

## Πώς σχεδιάζω ένα διανυσματικό επίπεδο για τον βασικό χάρτη;
`DrawVectorLayer` αποδίδει ένα διανυσματικό επίπεδο στον χάρτη χρησιμοποιώντας έναν συγκεκριμένο συμβολιστή.  
Η κλάση `Map` παρέχει τη μέθοδο `DrawVectorLayer` που διαβάζει ένα shapefile και αποδίδει κάθε χαρακτηριστικό χρησιμοποιώντας στυλ `SimpleFill`. Μπορείτε να προσαρμόσετε το χρώμα γεμίσματος, το πάχος του περιγράμματος και τη διαφάνεια ώστε να ταιριάζει με το οπτικό σας θέμα, καθώς και να εφαρμόσετε διαφάνεια ή μοτίβα γεμίσματος για πιο πλούσιες χαρτογραφικές επιδράσεις.

```csharp
var baseMapSymbolizer = new SimpleFill { FillColor = Color.Salmon, StrokeWidth = 0.75 };
map.Add(VectorLayer.Open(dataDir + "basemap.shp", Drivers.Shapefile), baseMapSymbolizer);
```

## Πώς φορτώνω GeoJSON και προσθέτω πόλεις στον χάρτη;
`FeatureBasedConfiguration` είναι ένας delegate που σας επιτρέπει να μορφοποιήσετε κάθε χαρακτηριστικό βάσει των ιδιοτήτων του.  
Η φόρτωση ενός αρχείου GeoJSON σας δίνει μια συλλογή σημειακών χαρακτηριστικών που αντιπροσωπεύουν πόλεις. Με τη μεταβίβαση ενός delegate `FeatureBasedConfiguration` μπορείτε να μορφοποιήσετε κάθε δείκτη πόλης βάσει ιδιοτήτων όπως ο πληθυσμός ή η περιοχή. Μπορείτε επίσης να φιλτράρετε χαρακτηριστικά ή να προσαρμόσετε το μέγεθος του συμβόλου δυναμικά για να αντανακλά επιπλέον διαστάσεις δεδομένων.

```csharp
var citiesSymbolizer = new SimpleMarker() { FillColor = Color.LightBlue };
citiesSymbolizer.FeatureBasedConfiguration = (feature, symbolizer) =>
{
    // Additional configuration logic can be added here.
};
map.Add(VectorLayer.Open(dataDir + "points.geojson", Drivers.GeoJson), citiesSymbolizer);
```

## Πώς εξάγω τον χάρτη ως αρχείο SVG;
`Map.Save` εξάγει τον αποδοθέντα χάρτη σε αρχείο στην καθορισμένη μορφή.  
Καλώντας το `Map.Save` με το όρισμα `SaveFormat.Svg` γράφει τα αποδοθέντα διανυσματικά δεδομένα σε αρχείο SVG στο δίσκο. Το αποτέλεσμα `cities_out.svg` μπορεί να ανοίξει απευθείας σε οποιονδήποτε σύγχρονο browser ή να επεξεργαστεί με εργαλεία όπως το Inkscape. Μπορείτε επίσης να καθορίσετε DPI ή να ενσωματώσετε CSS για περαιτέρω μορφοποίηση.

```csharp
map.Render(dataDir + "cities_out.svg", Renderers.Svg);
```

## Συνηθισμένα Προβλήματα και Λύσεις
- **Σφάλμα λείποντος αρχείου δεδομένων** – Επαληθεύστε ότι οι σχετικές διαδρομές προς το shapefile και το GeoJSON είναι σωστές· χρησιμοποιήστε απόλυτες διαδρομές κατά την αποσφαλμάτωση.  
- **Οι πόλεις δεν είναι ορατές** – Βεβαιωθείτε ότι το σύστημα αναφοράς συντεταγμένων του GeoJSON ταιριάζει με το CRS του βασικού χάρτη, ή επαναπροβάλετε τα δεδομένα χρησιμοποιώντας `Geometry.Transform`.  
- **Το SVG εμφανίζεται κενό** – Ελέγξτε ότι το χρώμα φόντου του χάρτη δεν είναι ορισμένο σε πλήρως διαφανές· ορίστε ένα ανοιχτό χρώμα για να επιβεβαιώσετε την απόδοση.
