## Vanliga frågor

**Q:** Hur skapar jag **create new shapefile**‑filer programatiskt?  
**A:** Använd `Shapefile`‑klassen i Aspose.GIS, specificera geometrityp, lägg till fält och skriv sedan funktioner. Den länkade “Create New Shapefile”‑handledningen visar den exakta koden.

**Q:** Vad är skillnaden mellan en File GDB och en shapefile?  
**A:** En File Geodatabase (File GDB) lagrar data i en mapp‑baserad struktur med rikare schemastöd, medan en shapefile är ett enkelt, brett kompatibelt format bestående av flera fasta‑längds‑filer.

**Q:** Hur kan jag **filter features by attribute** effektivt?  
**A:** Bygg ett `Query`‑objekt med ett WHERE‑villkor (t.ex. `SELECT * FROM layer WHERE Population > 1000`). “Filter Features by Attribute”‑guiden går igenom API‑et.

**Q:** Kan jag konvertera en shapefile till GeoJSON utan mellanfiler?  
**A:** Ja—läs in shapefilen i en `FeatureCollection` och anropa `ExportToGeoJson()` direkt. Se “Extract Features to GeoJSON”‑handledningen för ett komplett exempel.

**Q:** Behöver jag en speciell licens för att köra dessa exempel lokalt?  
**A:** En gratis utvärderingslicens fungerar för utveckling och testning. För produktionsdistributioner, köp en kommersiell licens från Aspose.
